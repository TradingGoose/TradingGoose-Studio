- The checkpoint table is treated as retired schema, not dormant runtime state. This branch removes both the Drizzle table export and the manual cleanup command, then records the drop in the migration chain so the DB contract and app code stay aligned.
- Copilot session persistence continues to live in `packages/db/schema/copilot.ts`. - No legacy backfill or compatibility path was added. The change is a clean retirement of the checkpoint table and its helper script.
