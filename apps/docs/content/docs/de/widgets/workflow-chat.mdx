Das Workflow-Chat-Widget bietet eine interaktive Chat-Oberfläche, um mit Ihren Workflow-Blöcken zu kommunizieren. Es ermöglicht konversationelle Interaktionen mit Ihren KI-Workflows.

## Übersicht

| Eigenschaft | Wert |
|-------------|------|
| **Schlüssel** | `workflow_chat` |
| **Kategorie** | Utility |
| **Icon** | MessageCircle |

## Funktionen

- **Interaktiver Chat**: Senden Sie Nachrichten und erhalten Sie Antworten von Ihrem Workflow
- **Auswahl der Ausgaben**: Wählen Sie, welche Workflow-Ausgaben im Chat angezeigt werden
- **Datei-Upload**: Hängen Sie Dateien an Ihre Nachrichten zur Verarbeitung an
- **Echtzeit-Streaming**: Sehen Sie Antworten, während sie generiert werden
- **Nachrichtenverlauf**: Sehen und scrollen Sie durch den Gesprächsverlauf

## Verwendung

Das Workflow-Chat-Widget ist nützlich für:

- Testen von Workflows mit chatbasierten Triggern
- Interaktion mit KI-Agenten in Ihrem Workflow
- Debuggen von Workflow-Antworten
- Demonstration der Workflow-Funktionen

### Nachrichten senden

1. Geben Sie Ihre Nachricht in das Eingabefeld ein
2. Optional: Fügen Sie über die Upload-Schaltfläche Dateien hinzu
3. Drücken Sie Enter oder klicken Sie auf Senden, um abzuschicken
4. Sehen Sie die gestreamte Antwort in Echtzeit

### Ausgabenauswahl

Sie können auswählen, welche Blockausgaben in der Chat-Antwort angezeigt werden. Das hilft, wenn Ihr Workflow mehrere Ausgabe-Blöcke hat und Sie sich auf bestimmte Ergebnisse konzentrieren möchten.

<Callout>
Dieses Widget unterstützt den **pairColor**-Mechanismus. Kombinieren Sie es mit einem Workflow-Editor, um mit dem Workflow zu chatten, den Sie gerade bearbeiten.
</Callout>
