Das Workflow Copilot-Widget bietet eine KI-gestützte Assistenzerfahrung, die speziell entwickelt wurde, um Ihnen beim Erstellen und Optimieren Ihrer Workflows zu helfen.

## Übersicht

| Eigenschaft | Wert |
|----------|-------|
| **Schlüssel** | `workflow_copilot` |
| **Kategorie** | Dienstprogramm |
| **Symbol** | Funkeln |

## Funktionen

- **KI-gestützte Unterstützung**: Erhalten Sie intelligente Hilfe beim Workflow-Design
- **Kontextbezogen**: Versteht Ihre aktuelle Workflow-Struktur
- **Block-Vorschläge**: Empfiehlt Blöcke basierend auf Ihren Zielen
- **Fehlerbehebung**: Hilft bei der Diagnose und Behebung von Workflow-Problemen
- **Best Practices**: Vorschläge zur Optimierung Ihrer Workflows

## Verwendung

Der Workflow Copilot kann Ihnen helfen bei:

- Entwerfen neuer Workflows von Grund auf
- Optimieren bestehender Workflows
- Verstehen von Block-Konfigurationen
- Beheben von Ausführungsproblemen
- Erlernen von TradingGoose-Funktionen

### Beispielinteraktionen

**Erstellen eines neuen Workflows:**
> "Erstelle einen Workflow, der ein GitHub-Repository überwacht und Slack-Benachrichtigungen für neue Issues sendet"

**Blöcke verstehen:**
> "Was macht der Router-Block und wann sollte ich ihn verwenden?"

**Fehlerbehebung:**
> "Warum schlägt mein Workflow beim API-Block fehl?"

<Callout>
Dieses Widget unterstützt den **pairColor**-Mechanismus. Kombinieren Sie es mit einem Workflow-Editor, um Unterstützung für den spezifischen Workflow zu erhalten, an dem Sie arbeiten.
</Callout>
