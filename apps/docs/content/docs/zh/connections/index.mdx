连接是允许数据在工作流中的各模块之间流动的通道。它们定义了信息如何从一个模块传递到另一个模块，使您能够创建复杂的多步骤流程。

<Callout type="info">
  正确配置的连接对于创建有效的工作流至关重要。它们决定了数据如何在系统中流动，以及各模块之间如何相互交互。
</Callout>

要创建连接，请从一个模块的输出端口拖动到另一个模块的输入端口。连接建立后，源模块产生的数据将通过连接标签在目标模块中可用。
