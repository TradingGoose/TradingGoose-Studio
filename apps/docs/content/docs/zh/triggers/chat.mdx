聊天触发器为您的工作流创建一个对话界面。将工作流部署为聊天应用后，用户可通过可分享的 URL 与其进行交互。每条消息都会启动一次新的工作流执行。

<Callout type="info">
  这是一个**基于部署**的触发器。用户通过托管的聊天 URL 与您的工作流进行交互。
</Callout>

### 输出模式

| 字段 | 类型 | 描述 |
|-------|------|-------------|
| `input` | string | 最新用户消息 |
| `conversationId` | string | 会话线程 ID |
| `files` | files | 可选上传文件 |

### 文件属性

| 属性 | 类型 | 描述 |
|----------|------|-------------|
| `<chat.files[0].url>` | string | 已签名的下载链接 |
| `<chat.files[0].name>` | string | 原始文件名 |
| `<chat.files[0].size>` | number | 文件大小（字节） |
| `<chat.files[0].type>` | string | MIME 类型 |

<Callout>
每个会话维护独立的线程。用户可以同时进行多个会话。
</Callout>
