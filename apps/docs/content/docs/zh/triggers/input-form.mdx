当工作流需要从编辑器以类型化字段启动时，请使用输入表单触发器。运行面板会精确显示您所定义的字段，从而确保工作流始终接收到规范的数据。

<Callout type="info">
  这是一个带有结构化输入的**手动**触发器。请定义您的模式并从编辑器运行。
</Callout>

## 配置

在输入格式构建器中添加字段。支持的类型：

- **string** — 文本值
- **number** — 数值
- **boolean** — 布尔值（真/假）
- **json** — JSON 对象

### 输出模式

| 字段 | 类型 | 描述 |
|-------|------|-------------|
| `<blockName.field>` | 可变 | 输入格式中定义的每个字段 |
| `<blockName.input>` | json | 完整的结构化输入负载 |

## 子工作流

输入表单触发器同样驱动工作流模块。当您嵌入子工作流时，映射步骤将显示子表单字段，以便您从父工作流中连接变量。

<Callout>
无需字段，只需一个快速运行按钮？请改用手动触发器。
</Callout>
