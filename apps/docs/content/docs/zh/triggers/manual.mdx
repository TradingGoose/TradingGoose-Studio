手动触发器会在您的工作流中添加一个运行按钮。当您希望立即执行工作流而无需收集额外数据时，可使用此触发器。

<Callout type="info">
  这是一个**手动**触发器。无需任何配置——只需点击运行即可。
</Callout>

## 配置

<ShowcaseCard>
  <BlockConfigPreview
    name="Manual Trigger"

无需任何配置。手动触发器没有输入字段，也不产生任何输出。

## 使用场景

- 快速完整性检查与冒烟测试
- 仅依赖内部变量或外部 API 调用的工作流
- 开发与调试

<Callout>
需要结构化输入？请改用输入表单触发器。
</Callout>
