Workflow Copilot 小部件提供了一种专为帮助您构建和优化工作流而设计的 AI 驱动助手体验。

## 概述

| 属性 | 值 |
|----------|-------|
| **键** | `workflow_copilot` |
| **类别** | 实用工具 |
| **图标** | Sparkles |

## 功能

- **AI 驱动的帮助**：为工作流设计提供智能帮助
- **上下文感知**：理解您当前的工作流结构
- **模块建议**：根据您的目标推荐模块
- **故障排除**：帮助诊断和修复工作流问题
- **最佳实践**：提供优化工作流的建议

## 用法

Workflow Copilot 可以帮助您：

- 从头设计新的工作流
- 优化现有工作流
- 理解模块配置
- 调试执行问题
- 学习 TradingGoose 功能

### 示例交互

**创建新工作流：**
> "创建一个监控 GitHub 仓库并为新问题发送 Slack 通知的工作流"

**理解模块：**
> "Router 模块的作用是什么，我应该在什么时候使用它？"

**调试：**
> "为什么我的工作流在 API 模块处失败？"

<Callout>
此小部件支持 **pairColor** 机制。将其与 Workflow Editor 配对，以获取您正在处理的特定工作流的帮助。
</Callout>
