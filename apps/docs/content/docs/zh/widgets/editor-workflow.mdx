工作流编辑器是 TradingGoose 中用于构建 AI 工作流的核心可视化界面。它提供了一个画布，您可以在其中拖放并连接模块，以创建强大的自动化流程。

## 概述

| 属性 | 值 |
|----------|-------|
| **键** | `editor_workflow` |
| **类别** | 编辑器 |
| **图标** | 工作流 |

## 功能

- **可视化画布**：直观的拖放界面，用于设计工作流
- **模块放置**：从模块库中添加模块到画布
- **连接管理**：在模块的输出和输入之间绘制连接
- **缩放和平移**：通过缩放和平移控件导航大型工作流
- **模块配置**：点击模块以配置其设置
- **实时验证**：即时反馈连接兼容性

## 用法

工作流编辑器通常是您工作区中的主要小部件，提供设计和修改 AI 工作流的可视化环境。

### 基本操作

1. **添加模块**：从侧边栏拖动模块或使用上下文菜单
2. **连接模块**：从输出端口点击并拖动到输入端口
3. **配置模块**：点击模块以打开其配置面板
4. **删除元素**：选择后按 Delete 键，或使用上下文菜单
5. **撤销/重做**：使用 Ctrl+Z / Ctrl+Shift+Z 进行历史导航

<Callout>
此小部件支持 **pairColor** 机制，可与其他工作流小部件（如控制台、聊天和变量）同步。
</Callout>
