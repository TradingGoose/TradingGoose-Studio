## 概述

| 属性 | 值 |
|----------|-------|
| **键** | `editor_skill` |
| **类别** | 编辑器 |

技能编辑器允许您修改工作区技能的名称、描述和指令内容。技能是可复用的指令集，智能体可在运行时加载以获得专项行为能力。使用 [技能列表](/utilities/skills) 组件浏览和创建技能；编辑器是您编写技能内容的地方。

## 功能

- **名称字段** 接受 kebab-case 格式的标识符（例如 `market-research`）。这是智能体加载技能时使用的键名。编辑器在保存时会验证该名称在工作区内的唯一性。
- **描述字段** 提供该技能帮助智能体执行任务的简短说明。
- **指令文本区域** 是一个自由格式的文本区域（最多 50,000 个字符），用于以纯文本或 Markdown 编写完整的技能指令。文本区域使用等宽字体以便于编辑。
- **保存按钮** 位于标题栏中，用于持久化所有更改。保存时会执行验证：名称必须为非空的 kebab-case 格式，描述和内容不得为空，且名称在工作区内必须唯一。
- **技能选择器** 位于标题栏中，用于在各技能之间切换。通过颜色链接关联后，选择器将与技能列表组件同步。

## 使用方法

1. 从组件选择器将该组件添加到工作区。
2. 从标题下拉菜单中选择一个技能，或从 [技能列表](/utilities/skills) 组件创建新技能。
3. 编辑 **名称**、**描述** 和 **指令** 字段。
4. 点击 **保存** 按钮以持久化您的更改。

### 验证规则

- **名称** 必须使用短横线命名法（小写字母、数字和连字符）。示例：`market-research`。
- **名称** 必须唯一——同一工作区内不能有两个同名技能。
- **描述** 和 **说明** 均为必填项，不能为空。

<Callout>
此小部件支持 **pairColor** 机制。为技能编辑器和 [技能列表](/utilities/skills) 小部件分配相同颜色，以便在列表中选择技能时自动在编辑器中打开该技能。
</Callout>

## 配置

所选技能以 `skillId` 的形式存储在小部件参数中，并在关联时通过配对颜色存储进行同步。
