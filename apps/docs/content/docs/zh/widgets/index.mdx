---
title: 组件
description: 可在工作区面板中自由排列的仪表盘 UI 组件。
---

import { Card, Cards } from 'fumadocs-ui/components/card'

组件是构成工作区的仪表盘 UI 元素。每个组件占据工作区布局中的一个面板，为特定任务提供专注的操作界面——编辑工作流、监控执行状态、绘制市场行情图表或管理工作区资源。您可以自由地重新排列、调整大小和切换组件，以构建适合您需求的布局。

<Cards>
  <Card title="Copilot" href="/widgets/copilot">

  </Card>
</Cards>

## 工作流

用于构建、运行和与工作流交互的组件。

<Cards>
  <Card title="工作流编辑器" href="/widgets/editor-workflow">
    通过连接模块来构建和编辑工作流的画布界面。
  </Card>
  <Card title="工作流列表" href="/widgets/list-workflow">
    支持拖放操作的文件夹树，用于浏览和整理工作流。
  </Card>
  <Card title="工作流聊天" href="/widgets/workflow-chat">
    在执行过程中与工作流模块进行交互的聊天界面。
  </Card>
  <Card title="工作流控制台" href="/widgets/workflow-console">
    实时执行控制台，显示日志和流式输出。
  </Card>
  <Card title="工作流变量" href="/widgets/workflow-variables">
    查看和编辑所选工作流的变量。
  </Card>
</Cards>
