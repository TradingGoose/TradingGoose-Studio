Workflow Chat 小部件提供交互式聊天界面，用于与你的工作流模块沟通，让你可以以对话方式与 AI 工作流互动。

## 概述

| 属性 | 值 |
|------|----|
| **键** | `workflow_chat` |
| **类别** | 实用工具 |
| **图标** | MessageCircle |

## 功能

- **交互式聊天**：向工作流发送消息并接收响应
- **输出选择**：选择在聊天中显示哪些工作流输出
- **文件上传**：为消息附加文件以供处理
- **实时流式传输**：在生成时实时查看响应
- **消息历史**：查看并滚动浏览会话历史

## 用途

Workflow Chat 小部件适用于：

- 测试具有聊天触发器的工作流
- 与工作流中的 AI 代理交互
- 调试工作流响应
- 演示工作流能力

### 发送消息

1. 在输入框中输入你的消息
2. 可选：使用上传按钮附加文件
3. 按 Enter 或点击发送提交
4. 实时查看流式响应

### 输出选择

你可以选择在聊天响应中显示哪些模块输出。当工作流有多个输出模块且你想聚焦于特定结果时，这会很有帮助。

<Callout>
该小部件支持 **pairColor** 机制。将它与 Workflow Editor 配对，可与当前正在编辑的工作流聊天。
</Callout>
