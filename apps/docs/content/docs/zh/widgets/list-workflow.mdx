## 概述

| 属性 | 值 |
|----------|-------|
| **键** | `list_workflow` |
| **类别** | 列表 |

工作流列表小部件在您的工作区中显示所有工作流的可浏览列表。

## 功能

- **浏览**：查看工作区中的所有工作流
- **搜索与筛选**：快速找到所需工作流
- **新建**：直接从列表中启动新工作流
- **在编辑器中打开**：点击即可在编辑器小部件中打开任意工作流

## 使用方法

将此小部件与工作流编辑器一起添加，以管理您的工作流。
