工作流变量小部件提供了一个专用界面，用于查看和编辑工作流中定义的变量。

## 概述

| 属性 | 值 |
|----------|-------|
| **键** | `workflow_variables` |
| **类别** | 实用工具 |
| **图标** | 大括号 |

## 功能

- **变量列表**：一目了然地查看所有工作流变量
- **内联编辑**：直接编辑变量值
- **类型显示**：查看每个变量的数据类型
- **添加/删除**：创建新变量或删除现有变量
- **搜索**：按名称筛选变量

## 用途

工作流变量小部件适用于：

- 检查工作流变量的当前状态
- 修改变量值以进行测试
- 理解工作流中的数据流
- 管理工作流范围的配置

### 变量类型

工作流变量可以包含不同类型的数据：

| 类型 | 描述 |
|------|-------------|
| **字符串** | 文本值 |
| **数字** | 数值 |
| **布尔值** | 真/假值 |
| **对象** | JSON 对象 |
| **数组** | 值的列表 |

### 编辑变量

1. 点击变量以选择它
2. 在编辑器中修改值
3. 更改会自动保存
4. 工作流在下次执行时使用更新后的值

### 创建变量

1. 点击“添加变量”按钮
2. 输入变量名称
3. 选择变量类型
4. 设置初始值（可选）

<Callout>
此小部件支持 **pairColor** 机制。将其与工作流编辑器配对，以查看您当前正在编辑的工作流的变量。
</Callout>
