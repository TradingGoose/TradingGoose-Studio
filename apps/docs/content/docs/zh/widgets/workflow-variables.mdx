## 概述

| 属性 | 值 |
|----------|-------|
| **键** | `workflow_variables` |
| **类别** | 实用工具 |

工作流变量组件为您提供一个专用面板，用于查看和编辑工作流中定义的变量。它通过实时套接字连接到工作流存储，因此支持协同操作——在此处所做的编辑会实时同步至编辑器，反之亦然。

## 功能

- **实时变量列表** -- 通过协同工作流存储实时显示当前工作流的所有变量
- **内联编辑** -- 直接在面板中编辑变量名称、类型和值；更改立即同步至所有已连接的客户端
- **添加变量** -- 标题栏中的 **+** 按钮会触发自定义 `WORKFLOW_VARIABLES_ADD_EVENT` 事件，在工作流中创建一个新的空字符串变量；该事件的作用域限定于正确的面板和频道，因此仅影响目标组件实例
- **协同同步** -- 底层使用 `useCollaborativeWorkflow`，确保添加和编辑操作广播至所有查看同一工作流的用户
- **工作流选择器** -- 标题栏下拉菜单允许在工作流之间切换；与编辑器关联时支持配对颜色同步

## 使用方法

### 添加组件

打开组件选择器并选择 **工作流变量**。它将在当前工作区中加载活动工作流的变量列表。

### 标题栏布局

- **居中** -- 工作流选择器下拉菜单
- **右侧** -- 添加变量按钮（**+**）

### 核心交互

1. 浏览变量列表，查看名称、类型和当前值
2. 点击变量可内联编辑其值
3. 点击标题栏中的 **+** 按钮可追加新变量（默认类型为 `string`，值为空）
4. 更改自动保存，并实时同步至其他协作者

### 配对颜色同步

分配配对颜色后，组件将跟随配对编辑器中选定的工作流。在编辑器中更改工作流会刷新变量列表。当配对颜色为 `gray` 时，您可通过标题栏下拉菜单独立选择要检查的工作流。
