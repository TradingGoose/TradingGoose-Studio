## 概述

| 属性 | 值 |
|----------|-------|
| **键** | `copilot` |
| **类别** | 实用工具 |

Copilot 组件是一个覆盖整个工作区的 AI 助手。它可以管理工作流、MCP 服务器、技能、自定义工具、知识库、部署、变量等。它为每个频道维护独立的聊天记录，并支持多个并发会话。

## 功能特性

- **覆盖整个工作区的 AI 助手** — 不局限于工作流。Copilot 可以创建和编辑 MCP 服务器、技能、自定义工具，查询知识库，部署工作流，设置变量以及运行工作流
- **上下文感知** — 当与编辑器面板（工作流、指标、技能、MCP 或自定义工具）配对使用时，Copilot 会自动包含该上下文，从而能够引用和修改正确的资源
- **持久化聊天记录** — 会话按频道存储，并在历史下拉菜单中按时间分组（今天、昨天、本周、更早）
- **多会话支持** — 通过标题栏中的 **+** 创建新会话；在不同会话间切换时不会丢失之前的对话
- **面板专属频道** — 通过配对颜色绑定后，每个面板拥有独立频道，使多个 Copilot 组件能够独立运行

## 使用方法

### 添加组件

打开组件选择器并选择 **Copilot**。它将初始化一个限定于当前工作区上下文的聊天会话。

### 标题栏布局

- **左侧** — 聊天记录下拉菜单；点击可浏览并切换历史会话
- **右侧** — 新建聊天按钮（**+**），用于开启新的对话

### 主要交互操作

1. 在输入区域输入问题或指令并发送
2. Copilot 流式返回响应，并根据需要执行工具调用以管理工作区资源
3. 打开历史记录下拉菜单，查看或删除之前的对话
4. 点击 **+** 新建聊天线程，同时保留旧线程的访问入口

### 配对颜色同步

当分配了配对颜色后，Copilot 将跟随配对编辑器的上下文——无论是工作流、指标、技能、MCP 服务器还是自定义工具。每个面板仍拥有独立的通道，因此同一配对颜色下的多个 Copilot 组件可独立运行。
