## 概览

| 属性 | 值 |
|----------|-------|
| **键** | `editor_indicator` |
| **类别** | 编辑器 |

指标编辑器是您编写和完善自定义 PineTS 指标的工作区。它提供功能完整的 Monaco 代码编辑器，支持 TypeScript，内置 PineTS 全局变量速查表，并通过服务端验证功能让您在保存前校验脚本。使用 [指标列表](/indicators) 小部件可浏览和创建指标；编辑器是编写指标逻辑的主要工作区。

## 功能特性

- **Monaco 代码编辑器**：支持 TypeScript 语法高亮、自动补全，并自动加载 PineTS 类型定义。
- **速查表工具栏**：按类别分组（数据、技术分析、输入、绘图等）。悬停任意符号可查看其签名、描述及使用示例。
- **验证按钮**：在服务端对指标进行模拟数据运行，报告生成的绘图数量和标记数量，以及任何关于不支持样式或绘图类型的警告。
- **保存按钮**：持久化保存 PineTS 代码，并自动从脚本中的 `input.*` 调用推断 `inputMeta`，以便图表能够渲染输入控件。
- **魔法棒提示栏**：允许您用自然语言描述指标逻辑，并将生成的 PineTS 代码直接流式传输至编辑器。
- **环境变量下拉框**：当您输入 `{{` 模式时触发，允许您在指标代码中引用工作区密钥。
- **指标选择器**：位于顶部导航栏，可在当前工作区的各指标之间切换。通过颜色链接配对后，选择器将与指标列表小部件同步。

## 使用方法

1. 从小部件选择器将该小部件添加到工作区。
2. 从顶部下拉菜单选择现有指标，或先在 [指标列表](/indicators) 小部件中创建一个。
3. 在编辑器中编写 PineTS 代码。直接使用全局变量 —— `close`、`ta.sma`、`input.int`、`plot` —— 无需 `$` 前缀。
4. 点击 **验证** 按钮（勾选图标）以执行服务端预运行。修复所有出现的错误或警告。
5. 点击 **保存** 按钮以持久化指标。图表将在下次渲染时加载更新后的代码。

### 键盘快捷键

- **魔棒切换**：点击编辑器边栏中的魔棒图标，可打开或关闭 AI 提示栏。

<Callout>
此小部件支持 **pairColor** 机制。为指标编辑器与 [指标列表](/indicators) 小部件分配相同颜色后，在列表中选择某个指标将自动在编辑器中将其打开。
</Callout>

## 配置

除指标选择器外，该小部件不提供其他面向用户的配置项。所选指标以 `pineIndicatorId` 的形式存储于小部件参数中，并在关联时通过配对颜色存储进行同步。
