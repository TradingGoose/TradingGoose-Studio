## 概述

| 属性 | 值 |
|----------|-------|
| **键** | `workflow_console` |
| **类别** | 实用工具 |

工作流控制台组件以实时日志的形式展示工作流执行过程中发生的所有事件。每条日志条目均标注了产生该条目的模块名称、执行状态及时间戳。该组件可响应面板宽度变化并自动调整布局。

## 功能

- **实时日志流** —— 随着每个模块的执行，日志条目即时显示，包含模块名称、类型、状态及输出内容
- **按状态和模块筛选** —— 通过筛选弹窗可切换 `error` 和 `info` 状态的显示与隐藏，并可按名称勾选或取消勾选各模块；筛选图标会高亮显示已启用的筛选条件
- **按时间排序** —— 通过表头按钮在时间升序与降序之间切换
- **结构化视图** —— 切换后将日志条目以格式化 JSON 样式布局呈现，便于检查复杂输出对象
- **文本换行** —— 控制较长日志行是换行显示还是水平滚动
- **导出为 CSV** —— 将当前（已筛选的）控制台条目下载为 CSV 文件
- **清空控制台** —— 从控制台存储中删除当前工作流的所有日志条目
- **工作流选择器** —— 通过表头下拉菜单切换工作流；在已关联的情况下支持配对颜色同步

## 使用方法

### 添加组件

打开组件选择器并选择 **工作流控制台**。它将加载当前工作区中活动工作流的日志条目。

### 头部布局

- **居中** -- 工作流选择下拉菜单
- **右侧** -- 工具栏，包含筛选、排序、结构化视图、自动换行、导出 CSV 及清空按钮（全部分组）

### 关键交互操作

1. 从编辑器或聊天中运行工作流，观察条目实时流入控制台
2. 打开筛选弹出框，将结果缩小至特定代码块或仅显示错误条目
3. 切换结构化视图，以便更轻松地读取 JSON 数据载荷
4. 当条目包含较长的单行字符串时，切换自动换行功能
5. 点击下载按钮，将筛选后的条目导出为 CSV 文件以供离线分析
6. 点击垃圾桶按钮，清空当前工作流的控制台

### 配对颜色同步

当分配了配对颜色后，控制台将自动跟随配对编辑器中所选的工作流。当配对颜色为 `gray` 时，您可以通过头部下拉菜单独立选择要监控的工作流。
