工作流控制台小部件提供工作流执行的实时视图，显示日志、调试信息以及工作流运行时的流式输出。

## 概述

| 属性 | 值 |
|----------|-------|
| **键** | `workflow_console` |
| **类别** | 实用工具 |
| **图标** | 活动 |

## 功能

- **实时日志**：实时查看执行日志
- **流式输出**：实时查看块输出
- **执行状态**：监控工作流执行的当前状态
- **错误显示**：快速识别和调试错误
- **日志过滤**：按类型、块或严重性过滤日志
- **清空控制台**：重置控制台输出

## 用法

工作流控制台适用于：

- 调试工作流执行
- 监控长时间运行的工作流
- 查看中间块输出
- 识别错误和瓶颈

### 日志类型

| 类型 | 描述 |
|------|-------------|
| **信息** | 一般执行信息 |
| **调试** | 详细的调试信息 |
| **警告** | 非关键性问题 |
| **错误** | 执行错误和失败 |
| **输出** | 块输出数据 |

### 查看执行流程

控制台显示工作流的执行路径：

1. **开始**：工作流执行开始
2. **块进入**：每个块开始处理
3. **块输出**：每个块的结果
4. **块退出**：块完成
5. **结束**：工作流执行完成

<Callout>
此小部件支持 **pairColor** 机制。将其与工作流编辑器配对，可以查看您正在编辑的工作流的日志。
</Callout>
