工作流列表小部件以分层文件夹树的形式显示所有工作流，帮助您组织、导航和管理工作流集合。

## 概览

| 属性 | 值 |
|----------|-------|
| **键** | `workflow_list` |
| **类别** | 列表 |
| **图标** | ListTree |

## 功能

- **文件夹树**：分层组织工作流
- **拖放操作**：在文件夹之间移动工作流
- **快速导航**：点击打开工作流
- **上下文菜单**：右键单击以获取更多操作
- **搜索**：按名称筛选工作流
- **新建**：添加新的工作流和文件夹

## 用法

工作流列表小部件与侧边栏导航相呼应，提供了一个专门用于管理工作流的视图。

### 组织

- **文件夹**：创建文件夹以分组相关工作流
- **拖放操作**：通过拖动将项目移动到新位置
- **重命名**：双击或使用上下文菜单进行重命名
- **删除**：移除工作流或空文件夹

### 上下文菜单操作

右键单击工作流或文件夹以访问：

| 操作 | 描述 |
|--------|-------------|
| **打开** | 在编辑器中打开工作流 |
| **重命名** | 更改名称 |
| **复制** | 创建副本 |
| **移动到** | 移动到另一个文件夹 |
| **删除** | 移除项目 |

### 选择工作流

点击工作流以选择它。当与使用相同颜色的其他小部件配对时，所选工作流将在这些小部件中显示。

<Callout>
此小部件支持 **pairColor** 机制。当您在此列表中选择一个工作流时，其他配对的小部件将自动切换以显示该工作流。
</Callout>
