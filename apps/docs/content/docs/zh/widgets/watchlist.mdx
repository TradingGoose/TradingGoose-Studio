## 概览

| 属性 | 值 |
|----------|-------|
| **键** | `watchlist` |
| **类别** | 数据 |

自选列表组件允许您创建、整理和监控交易品种列表，并提供实时报价数据。它显示最新价格、涨跌额和涨跌幅列，支持拖放排序、可折叠分组以及内联编辑交易品种。

## 功能特性

- **多个自选列表** -- 在工作区内创建、重命名和删除自选列表。标题栏中的列表选择器可在不同列表间切换。系统自选列表（自动生成）不可删除。
- **实时行情列** -- 每行显示**最新价**、**涨跌额**和**涨跌幅**，正值显示为绿色，负值显示为红色。行情自动刷新，也可通过标题栏按钮手动刷新。
- **分组** -- 将交易品种归类至可折叠的分组标题下。分组可内联重命名，并可独立展开或折叠。
- **拖放排序** -- 使用鼠标、触摸或键盘（通过可排序键盘传感器的方向键）对交易品种和分组进行排序。条目可在分组之间移动。
- **内联编辑交易品种** -- 双击某行可打开内联标的选择器，直接更改该条目的交易品种，无需先删除再重新添加。
- **搜索并添加交易品种** -- 标题栏中央控件提供标的搜索字段。选择交易品种后点击确认按钮，即可将其追加至当前自选列表。
- **导入 / 导出** -- 通过列表操作菜单从 JSON 文件导入交易品种，或将当前自选列表导出为 JSON 格式。
- **数据源选择** -- 从标题栏左侧控件中选择行情数据源。可通过设置弹出窗口配置数据源特定参数及 API 凭据。
- **配对颜色联动** -- 当组件被分配配对颜色后，点击某行将把所选标的广播至共享相同颜色的其他组件（例如[数据图表](/widgets/data-chart)），当前联动的交易品种将被高亮显示。

## 使用方法

将观察列表添加到工作区面板。顶部栏分为三个组：

- **左侧** -- 数据源设置按钮、数据源选择下拉菜单及手动刷新按钮。
- **中间** -- 标的搜索输入框及添加按钮（用于追加所选标的）。
- **右侧** -- 观察列表选择下拉菜单（切换列表、重命名或删除）及列表操作菜单（创建观察列表、添加分组、导入、导出、删除）。

单击某行以选中它。当该组件与数据图表共享相同的配对颜色时，单击操作还会在图表中加载对应标的。双击可进入编辑模式，直接在行内修改标的。每行右侧的控件允许删除单个条目，分组标题支持重命名和删除操作。

<Callout type="info">
为观察列表和数据图表分配相同的配对颜色，这样单击列表中的标的即可立即加载其图表。
</Callout>

## 配置

| 参数 | 说明 |
|-----------|-------------|
| `watchlistId` | 当前所选观察列表的 ID。 |
| `provider` | 用于获取行情报价的市场数据源。 |
| `providerParams` | 数据源专属参数（如交易所、市场类型）。 |
| `auth` | 所选数据源的可选 API 密钥/私钥。 |
| `runtime.refreshAt` | Unix 纪元时间戳，更新时触发行情刷新。 |
