备注块是一种仅用于可视化的注释，存在于工作流画布上。它不会执行，没有输入或输出，且完全被执行器忽略。可用于记录决策、为协作者留下说明，或对工作流中的复杂部分进行解释。

## 概述

备注块仅用于组织管理目的：

- **记录设计决策** -- 解释为何选择某种分支策略或模型。
- **留下说明** -- 告知协作者如何配置环境变量，或说明工作流某个部分的功能。
- **嵌入参考资料** -- 粘贴 YouTube 链接，链接将直接在画布上渲染为嵌入式视频。

<Callout type="info">
  备注块对工作流执行没有任何影响。它们不产生任何输出，也无法与其他块连接。工作流运行时，备注块会被完全剔除。
</Callout>

## 配置

唯一的字段是一个 Markdown 文本区域。可以输入纯文本、使用 Markdown 格式（标题、列表、粗体、代码块），或粘贴 YouTube URL 以嵌入视频。

## 最佳实践

- **将备注放置在其所描述的代码块附近**，以便上下文一目了然。
- **使用 Markdown 标题和列表**，使较长的备注易于浏览。
- **优先使用备注而非代码块中的注释** -- 备注在画布层级即可直接查看，无需打开代码块的配置。
