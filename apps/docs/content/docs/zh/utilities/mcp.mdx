[模型上下文协议](https://modelcontextprotocol.com/)（MCP）是一种开放标准，允许 AI 智能体安全地连接到外部工具和数据源。TradingGoose 实现了一个 MCP 客户端，使用 JSON-RPC 2.0 协议，并支持跨多个规范版本（`2025-06-18`、`2025-03-26`、`2024-11-05`）的协议版本协商。

## 支持的传输方式

您注册的每个 MCP 服务器均使用以下三种基于 HTTP 的传输模式之一：

| 传输方式 | 描述 |
|-----------|-------------|
| **HTTP** | 标准 HTTP POST，返回 JSON 响应。适用于简单的请求/响应工具。 |
| **SSE** | 服务器发送事件。服务器可能以 `text/event-stream` 形式返回流式结果。 |
| **Streamable HTTP** | 最新的 MCP 传输方式（规范 2025-03-26）。支持通过 `Mcp-Session-Id` 请求头进行会话管理，以及 JSON 与 SSE 之间的内容类型协商。 |

所有三种传输方式均由同一底层客户端处理——transport 字段控制连接的标识方式，但客户端会根据服务器的 `Content-Type` 响应头自动协商响应格式。

## 添加 MCP 服务器

<Steps>
<Step>
### 打开 MCP 编辑器小部件

将 **MCP 编辑器**小部件添加到您的仪表板，或导航至**工作区设置 > MCP 服务器**。有关小部件界面的详细信息，请参阅 [MCP 编辑器小部件文档](/widgets/editor-mcp)。
</Step>
<Step>
### 填写服务器配置

每个服务器需要以下信息：

- **名称** -- 便于识别的标签。
- **传输方式** -- `http`、`sse` 或 `streamable-http` 之一。
- **URL** -- 服务器端点（例如 `https://mcp.example.com/v1`）。
- **请求头**（可选）-- 自定义请求头，例如 `Authorization`。支持环境变量插值（详见下文）。
- **超时**（可选）-- 请求超时时间，单位为毫秒，默认值为 30 000 毫秒。
- **重试次数**（可选）-- 重试尝试次数，默认值为 3 次。
</Step>
<Step>
### 保存并测试

点击**保存**，然后使用 MCP 编辑器工具栏中的**测试连接**按钮。客户端将执行完整的初始化握手流程——包括版本协商、能力交换以及 `initialized` 通知——以确认服务器可达。
</Step>
</Steps>

## 认证与环境变量

您可以在服务器 URL 和请求头中使用双花括号语法引用工作区或个人环境变量：

```
Authorization: Bearer {{MY_API_KEY}}
```

运行时，服务会根据您的有效解密环境解析这些占位符。如果引用的变量缺失，请求将失败并返回一条描述性错误信息，列出未解析的变量名称。

<Callout type="warn">
切勿将密钥直接硬编码在服务器配置中。请始终将敏感值存储为环境变量，并使用 `{{VAR_NAME}}` 语法进行引用。
</Callout>

## 工具发现

当您连接到服务器时，客户端会发送 `tools/list` JSON-RPC 请求。响应将被解析为工具列表，每个工具包含：

- **name** —— 该服务器上工具的唯一标识符。
- **description** —— 工具的功能描述。
- **inputSchema** —— 描述预期参数的 JSON Schema。

已发现的工具按工作区缓存，采用 LRU 淘汰策略（最多 1000 条记录）。缓存在过期时自动刷新，或可从 MCP 编辑器工具栏手动强制刷新。

## 在 Agent 块中使用 MCP 工具

一旦服务器注册并启用，其工具将与内置工具一同显示在任意 **Agent** 块的工具选择器中。

<Steps>
<Step>
### 选择 MCP 工具

打开 Agent 块并展开**工具**部分。MCP 工具按服务器名称分组列出，选择您所需的工具。
</Step>
<Step>
### 运行 Agent

在执行过程中，代理在相应的 MCP 服务器上调用 `tools/call`，传递工具名称和参数。结果以结构化内容（文本、图像或资源）的形式返回，并反馈至代理的上下文中。
</Step>
</Steps>

每次工具执行都经过授权与审计框架的处理。客户端在分发调用前，会根据安全策略（拦截的来源、速率限制）对服务器进行检查，并记录审计追踪日志。

## 安全性

MCP 客户端默认执行以下几项安全措施：

- **授权门控** -- 每次工具执行在分发前均会根据当前安全策略进行检查。
- **来源白名单** -- 服务器 URL 来源会自动添加至允许来源列表。
- **速率限制** -- 可配置的每小时工具执行次数上限（默认值：1000）。
- **审计日志** -- 每项授权决策均标记唯一的审计 ID，以便追溯。
- **拦截来源** -- 位于拦截列表中的服务器将被立即拒绝。

## 故障排查

<Cards>
<Card title="服务器无法连接">
请确认 URL 是否正确且服务器正在运行。客户端会自动尝试带尾部斜杠和不带尾部斜杠两种形式，路径问题可自动处理。请查看浏览器控制台以获取详细的连接错误信息。
</Card>
<Card title="工具未显示">
请确认服务器已**启用**且未被软删除。使用 MCP 编辑器中的**刷新工具**按钮强制清除缓存并重新发现工具。
</Card>
<Card title="缺少环境变量错误">
错误信息中会列出所有未解析的变量。请在工作区或个人环境设置中添加相应变量，然后重试。
</Card>
<Card title="超时错误">
请在 MCP 编辑器中增加服务器超时时间，默认值为 30 秒。响应较慢的服务器或返回大量数据的工具可能需要更长时间。
</Card>
</Cards>
