TradingGoose 中所有受支持的 PineTS 函数的完整参考，按命名空间组织。本页内容源自 PineTS 运行时类型定义。

## input（13 个函数）

| 函数 | 返回值 | 描述 |
|----------|---------|-------------|
| `input.int(defval, title, min?, max?, step?)` | `number` | 整数输入 |
| `input.float(defval, title, min?, max?, step?)` | `number` | 浮点数输入 |
| `input.bool(defval, title)` | `boolean` | 布尔值输入 |
| `input.string(defval, title)` | `string` | 字符串输入 |
| `input.color(defval, title)` | `string` | 颜色选择器 |
| `input.source(defval, title)` | `series` | 数据源选择器 |
| `input.timeframe(defval, title)` | `string` | 时间框架选择器 |
| `input.price(defval, title)` | `number` | 价格输入 |
| `input.time(defval, title)` | `number` | 时间戳输入 |
| `input.session(defval, title)` | `string` | 交易时段输入 |
| `input.symbol(defval, title)` | `string` | 交易品种输入 |

| `input.enum(defval, title, options)` | `T` | 下拉选择器 |

<Callout type="info">
  本参考文档涵盖 TradingGoose 的 PineTS 运行时中可用的函数。PineTS 基于开源 [PineTS 项目](https://github.com/nicholasgasior/PineTS)，支持 Pine Script v5/v6 API 的部分功能。`strategy.*`、`label.*`、`line.*`、`box.*` 和 `table.*` 等功能目前暂不支持。
