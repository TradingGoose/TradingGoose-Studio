## 绘图函数

| 函数 | 描述 |
|----------|-------------|
| `plot(series, title?, options?)` | 绘制折线、面积、直方图或其他序列样式 |
| `plotshape(series, title?, options?)` | 绘制形状标记（箭头、圆形、菱形等） |
| `plotchar(series, title?, char?, options?)` | 绘制字符标记 |
| `plotarrow(series, title?, options?)` | 绘制箭头标记 |
| `plotbar(open, high, low, close, title?, options?)` | 绘制 OHLC K线柱 |
| `plotcandle(open, high, low, close, title?, options?)` | 绘制蜡烛图 |
| `hline(price, title?, options?)` | 在固定价格水平处绘制水平线 |
| `fill(plot1, plot2, color?)` | 填充两个绘图之间的区域 |
| `bgcolor(color, offset?, options?)` | 设置K线柱的背景颜色 |
| `barcolor(color, offset?, options?)` | 设置K线柱颜色 |

## 绘图样式

使用 `plot.style_*` 常量控制数据的渲染方式：

| 样式 | 描述 |
|-------|-------------|
| `plot.style_line` | 连续折线（默认） |
| `plot.style_linebr` | 在 `na` 值处断开的折线 |
| `plot.style_stepline` | 阶梯线 |
| `plot.style_stepline_diamond` | 带菱形标记的阶梯线 |
| `plot.style_steplinebr` | 带断开的阶梯线 |
| `plot.style_area` | 线条下方的填充面积 |
| `plot.style_areabr` | 在 `na` 处断开的面积 |
| `plot.style_histogram` | 垂直直方图柱 |
| `plot.style_columns` | 柱状图 |
| `plot.style_circles` | 圆形标记 |
| `plot.style_cross` | 十字标记 |

### 线型样式（适用于 `plot` 和 `hline`）

| 样式 | 描述 |
|-------|-------------|
| `plot.linestyle_solid` | 实线 |
| `plot.linestyle_dashed` | 虚线 |
| `plot.linestyle_dotted` | 点线 |
| `hline.style_solid` | 水平实线 |
| `hline.style_dashed` | 水平虚线 |
| `hline.style_dotted` | 水平点线 |

## 示例

### 基本折线图

```javascript
indicator('EMA Crossover', { overlay: true });

const fast = ta.ema(close, 9);
const slow = ta.ema(close, 21);

plot(fast, 'Fast EMA', { color: '#2962FF' });
plot(slow, 'Slow EMA', { color: '#FF6D00' });
```

### 直方图

```javascript
indicator('MACD Histogram');

const [macdLine, signalLine, hist] = ta.macd(close, 12, 26, 9);

plot(hist, 'Histogram', {
  style: plot.style_histogram,
  color: hist > 0 ? '#26A69A' : '#EF5350'
});
plot(macdLine, 'MACD', { color: '#2962FF' });
plot(signalLine, 'Signal', { color: '#FF6D00' });
```

### 区间填充图

```javascript
indicator('Bollinger Bands', { overlay: true });

const [mid, upper, lower] = ta.bb(close, 20, 2);

const p1 = plot(upper, 'Upper', { color: '#F23645' });
plot(mid, 'Middle', { color: '#787B86' });
const p2 = plot(lower, 'Lower', { color: '#089981' });

fill(p1, p2);
```

### 形状标记

```javascript
// Plot buy/sell arrows at crossover points
if (ta.crossover(fast, slow)) {
  plotshape(close, 'Buy', {
    style: shape.triangleup,
    location: location.belowbar,
    color: '#00FF00',
    size: size.small
  });
}

if (ta.crossunder(fast, slow)) {
  plotshape(close, 'Sell', {
    style: shape.triangledown,
    location: location.abovebar,
    color: '#FF0000',
    size: size.small
  });
}
```

### 形状常量

| 常量 | 描述 |
|----------|-------------|
| `shape.triangleup` | 向上三角形 |
| `shape.triangledown` | 向下三角形 |
| `shape.circle` | 圆形 |
| `shape.cross` | 十字形 |
| `shape.xcross` | X形十字 |
| `shape.diamond` | 菱形 |
| `shape.square` | 正方形 |
| `shape.arrowup` | 向上箭头 |
| `shape.arrowdown` | 向下箭头 |
| `shape.flag` | 旗帜 |
| `shape.labelup` | 向上标签 |
| `shape.labeldown` | 向下标签 |

### 位置常量

| 常量 | 描述 |
|----------|-------------|
| `location.abovebar` | 柱形上方 |
| `location.belowbar` | 柱形下方 |
| `location.top` | 窗格顶部 |
| `location.bottom` | 窗格底部 |
| `location.absolute` | 精确数值处 |

### 尺寸常量

| 常量 | 描述 |
|----------|-------------|
| `size.tiny` | 极小 |
| `size.small` | 小 |
| `size.normal` | 正常 |
| `size.large` | 大 |
| `size.huge` | 超大 |
| `size.auto` | 自动 |

### 背景色与柱形颜色

```javascript
// Highlight bars based on condition
bgcolor(ta.crossover(fast, slow) ? '#00FF0020' : na);
barcolor(close > open ? '#26A69A' : '#EF5350');
```

<Callout type="info">
  颜色可以指定为十六进制字符串（例如，`'#FF0000'`），也可以使用 `color` 命名空间（例如，`color.red`）。在末尾附加两位十六进制数字可设置透明度（例如，`'#FF000080'` 表示 50% 不透明度）。
</Callout>
