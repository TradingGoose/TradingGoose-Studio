PineTS 提供内置数据序列，可在每个指标中直接使用，无需导入。这些序列代表当前交易品种的 OHLCV 数据、时间信息及K线状态。

## 价格数据 (OHLCV)

| 序列 | 类型 | 描述 |
|--------|------|-------------|
| `open` | `series<number>` | 每根K线的开盘价 |
| `high` | `series<number>` | 每根K线的最高价 |
| `low` | `series<number>` | 每根K线的最低价 |
| `close` | `series<number>` | 每根K线的收盘价 |
| `volume` | `series<number>` | 每根K线的成交量 |

### 衍生价格序列

| 序列 | 公式 | 描述 |
|--------|---------|-------------|
| `hl2` | `(high + low) / 2` | 中间价 |
| `hlc3` | `(high + low + close) / 3` | 典型价格 |
| `ohlc4` | `(open + high + low + close) / 4` | 平均价格 |

```javascript
// Use any price series in calculations
const typicalPrice = hlc3;
const sma = ta.sma(typicalPrice, 20);
plot(sma, 'SMA of Typical Price');
```

## 时间序列

| 序列 | 类型 | 描述 |
|--------|------|-------------|
| `openTime` | `series<number>` | K线开盘时间（毫秒） |
| `closeTime` | `series<number>` | K线收盘时间（毫秒） |

## K线信息

| 变量 | 类型 | 描述 |
|----------|------|-------------|
| `bar_index` | `series<number>` | 当前K线索引（从0开始） |
| `last_bar_index` | `series<number>` | 数据集中最后一根K线的索引 |
| `last_bar_time` | `series<number>` | 最后一根K线的时间戳 |

```javascript
// Only plot on the last 100 bars
if (bar_index > last_bar_index - 100) {
  plot(close, 'Recent Close');
}
```

## K线状态

`barstate` 对象提供有关当前K线在数据集中位置的标志位：

| 属性 | 类型 | 描述 |
|----------|------|-------------|
| `barstate.isfirst` | `boolean` | 第一根K线时为 True |
| `barstate.islast` | `boolean` | 最后一根K线时为 True |
| `barstate.ishistory` | `boolean` | 历史K线时为 True |
| `barstate.isrealtime` | `boolean` | 实时/流式K线时为 True |
| `barstate.isnew` | `boolean` | 新K线第一个Tick时为 True |
| `barstate.isconfirmed` | `boolean` | K线已确认（收盘）时为 True |
| `barstate.islastconfirmedhistory` | `boolean` | 最后一根已确认历史K线时为 True |

## 标的信息

`syminfo` 对象提供当前标的的详细信息：

| 属性 | 描述 |
|----------|-------------|
| `syminfo.ticker` | 标的名称（例如：`"BTCUSDT"`）|
| `syminfo.tickerid` | 完整代码 ID |
| `syminfo.description` | 标的描述 |
| `syminfo.type` | 资产类型（例如：`"crypto"`、`"stock"`）|
| `syminfo.currency` | 计价货币 |
| `syminfo.basecurrency` | 基础货币 |
| `syminfo.mintick` | 最小价格变动单位 |
| `syminfo.timezone` | 交易所时区 |

## 特殊值

| 值 | 描述 |
|-------|-------------|
| `na` | 表示"不可用"——类似于 `null`。用于检查某个值是否存在 |
| `nz(value, replacement?)` | 若 `value` 为 `na`，则返回 `replacement`（默认 `0`）|

```javascript
const prev = close[1];
// na check
if (!na(prev)) {
  const change = close - prev;
  plot(change, 'Change');
}

// nz shorthand — default to 0 if na
const safeChange = nz(close - close[1], 0);
```

## 全局常量

| 命名空间 | 描述 |
|-----------|-------------|
| `timeframe` | 当前时间周期信息（`timeframe.period`、`timeframe.multiplier`）|
| `order` | 排序顺序常量（`order.ascending`、`order.descending`）|
| `currency` | 货币代码常量（共 59 种货币，包括 `currency.BTC`、`currency.USD`）|
| `display` | 显示模式常量 |
| `shape` | `plotshape()` 的形状类型常量 |
| `location` | 位置常量（`location.abovebar`、`location.belowbar` 等）|
| `size` | 尺寸常量（`size.tiny`、`size.small`、`size.normal`、`size.large`、`size.huge`）|
| `format` | 格式常量（`format.price`、`format.volume`、`format.percent`）|
| `dayofweek` | 星期常量 |

<Callout type="info">
  所有数据序列均支持通过方括号语法访问历史数据：`close[1]` 返回上一根K线的收盘价，`close[10]` 返回10根K线前的数据。
</Callout>
