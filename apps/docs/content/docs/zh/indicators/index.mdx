指标是在 TradingGoose 中对市场数据运行的自定义技术分析脚本。它们由 [PineTS](https://github.com/nicholasgasior/PineTS) 提供支持——这是一个适用于 Pine Script 的 JavaScript/TypeScript 运行时，并扩展了 TradingGoose 特有的功能，如工作流触发器。

## 您可以构建什么？

- **技术指标** — RSI、MACD、布林带，或您自定义的计算逻辑
- **信号生成器** — 检测交叉、突破或自定义条件
- **工作流触发器** — 当指标条件满足时，使用 `trigger()` API 触发工作流执行
- **图表叠加层** — 直接在图表上绘制线条、形状和填充区域

## 工作原理

<Steps>
  <Step>
    **编写 PineTS 代码**：在指标编辑器中使用适配 JavaScript/TypeScript 的熟悉 Pine Script 语法
  </Step>
  <Step>
    **定义输入参数**：使用 `input.int()`、`input.float()` 等——这些将成为 UI 中的可配置参数
  </Step>
  <Step>
    **执行计算**：使用 57 个以上的内置技术分析函数（`ta.*`）、数学运算、数组等
  </Step>
  <Step>
    **数据可视化**：使用 `plot()`、`plotshape()`、`fill()`、`bgcolor()` 及其他图表渲染函数
  </Step>
  <Step>
    **触发工作流**：当条件满足时调用 `trigger()`——将指标与您的自动化流程相连接
  </Step>
</Steps>

## 快速示例

```javascript
indicator('RSI Signal', { overlay: false });

const length = input.int(14, 'RSI Length');
const overbought = input.int(70, 'Overbought');
const oversold = input.int(30, 'Oversold');

const rsi = ta.rsi(close, length);

plot(rsi, 'RSI', { color: '#2962FF' });
hline(overbought, { color: '#FF0000', linestyle: hline.style_dashed });
hline(oversold, { color: '#00FF00', linestyle: hline.style_dashed });

// Trigger a workflow when RSI crosses below oversold
if (ta.crossunder(rsi, oversold)) {
  trigger('rsi_oversold', {
    condition: ta.crossunder(rsi, oversold),
    input: 'RSI crossed below oversold threshold',
    signal: 'long',
    position: 'belowBar',
    color: '#00FF00'
  });
}
```

## 内置指标

TradingGoose 内置了 80 个以上预构建指标，涵盖动量、趋势、波动率和成交量分析。这些指标既可作为即用型工具，也可作为编写自定义指标的参考示例。

<Callout type="info">
  PineTS 基于 Pine Script v5/v6 语法。如果您熟悉 TradingView 的 Pine Script，将会感到得心应手——主要区别在于使用 JavaScript 变量声明（`const`、`let`、`var`），而非 Pine Script 的隐式声明。
</Callout>
