## 直接 API 访问

在 SDK 开发期间，您可以直接使用 REST API 与 TradingGoose 集成：

```bash
curl -X POST \
  https://tradinggoose.ai/api/workflows/WORKFLOW_ID/execute \
  -H 'Content-Type: application/json' \
  -H 'X-API-Key: YOUR_KEY' \
  -d '{"input": "your data"}'
```

请参阅 [执行 API](/execution/api) 文档，了解完整的端点说明、流式传输及 Webhook 回调。
