## 过渡期间

直接使用 [执行 API](/execution/api) 配合 fetch：

```typescript
const response = await fetch(
  "https://tradinggoose.ai/api/workflows/WORKFLOW_ID/execute",
  {
    method: "POST",
    headers: {
      "Content-Type": "application/json",
      "X-API-Key": "YOUR_KEY",
    },
    body: JSON.stringify({ input: "your data" }),
  }
);

const result = await response.json();
```

有关使用 Server-Sent Events 处理流式响应的信息，请参阅 API 触发器文档中的[流式响应部分](/triggers/api#streaming-responses)。
