## 状态

Python SDK 正在构建中，旨在为执行 TradingGoose 工作流、管理工作区资源及处理流式响应提供原生 Python 接口。

## 过渡期间

可直接通过任意 HTTP 客户端使用 [执行 API](/execution/api)：

```python
import requests

response = requests.post(
    "https://tradinggoose.ai/api/workflows/WORKFLOW_ID/execute",
    headers={
        "Content-Type": "application/json",
        "X-API-Key": "YOUR_KEY",
    },
    json={"input": "your data"},
)

result = response.json()
```

有关流式响应，请参阅 API 触发器文档中的 [流式响应部分](/triggers/api#streaming-responses)。
