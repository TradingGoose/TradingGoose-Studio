TradingGoose Studio 是一个面向技术性 LLM 交易的 AI 工作流平台。它将技术分析与大型语言模型相结合，为您的交易决策提供全方位支持——涵盖数据分析、市场研究、图表绘制、持仓监控及工作流自动化。

通过连接各类模块，在可视化画布上构建工作流——包括 AI 智能体、交易操作、技术指标、API 集成等。大多数工作流无需编写代码，同时完整支持 PineTS 脚本，用于自定义指标与策略开发。

## 可构建的功能

**交易自动化**
连接券商（Alpaca、Tradier、Robinhood），执行订单，监控持仓，并基于 AI 分析与技术指标自动运行交易策略。

**技术分析与图表**
使用 PineTS 编写自定义指标，通过交互式K线图可视化 OHLCV 数据，管理带有实时行情的自选股列表，并在指标条件触发时自动启动工作流。

**AI 驱动的研究**
构建研究智能体，分析市场数据、扫描新闻资讯、处理金融文档，并使用来自 OpenAI、Anthropic、Google 等平台的模型生成研究报告。

**数据处理与集成**
连接 120+ 项服务——包括数据库、通信工具、CRM 系统及各类 API。通过条件判断、循环、并行执行与错误处理，编排复杂的数据管道。

**可视化工作流画布**
通过拖放模块构建工作流，以点击连接的方式集成 AI 模型、交易服务、数据库与 API。

**交易基础设施**
内置券商集成（Alpaca、Tradier、Robinhood）和市场数据提供商（Alpha Vantage、Finnhub、Yahoo Finance），支持实盘与模拟交易。

**PineTS 脚本**
使用 PineTS 编写自定义指标——这是一个兼容 Pine Script 的 JavaScript/TypeScript 运行时。通过 `trigger()` API 将指标信号连接至工作流自动化。

**可扩展性**
通过 MCP 服务器、自定义工具、可复用技能和知识库扩展平台功能。AI Copilot 帮助您在整个工作区中构建和管理所有内容。

**多种触发方式**
通过聊天界面、REST API、Webhook、定时任务、指标信号，或来自 Slack、GitHub 等服务的外部事件启动工作流。

## 内置集成

TradingGoose 连接 120+ 个服务：

- **交易**：Alpaca、Tradier、Robinhood、Kalshi、Polymarket
- **市场数据**：Alpha Vantage、Finnhub、Yahoo Finance
- **AI 模型**：OpenAI、Anthropic、Google、Groq、Cerebras、DeepSeek、Ollama（本地）
- **通讯**：Gmail、Slack、Teams、Telegram、WhatsApp、Discord
- **生产力**：Notion、Google Sheets、Airtable、Jira、Linear
- **开发**：GitHub、GitLab、浏览器自动化（Stagehand）
- **搜索与网络**：Google Search、Perplexity、Tavily、Exa、Firecrawl
- **数据库**：PostgreSQL、MySQL、MongoDB、Supabase、Pinecone、Qdrant

需要自定义功能？使用 [MCP 集成](/utilities/mcp) 连接任意外部服务，或构建[自定义工具](/utilities/custom-tools)。

## 部署选项

**云托管**：即将推出——提供托管基础设施，支持自动扩缩容与内置监控。

**自托管**：使用 Docker 在您自己的基础设施上部署，并通过 Ollama 支持本地 AI 模型，确保数据完全私密。
