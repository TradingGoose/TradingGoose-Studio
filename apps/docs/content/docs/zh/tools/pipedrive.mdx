| `status` | string | 否 | 仅获取具有特定状态的交易。可选值：open（进行中）、won（已赢得）、lost（已失去）。若省略，则返回所有未删除的交易 |

| `org_id` | string | 否 | 若提供，则仅返回与指定组织关联的交易 |
| `pipeline_id` | string | 否 | 若提供，则仅返回指定管道中的交易 |
| `updated_since` | string | 否 | 若设置，则仅返回此时间之后更新的交易。格式：2025-01-01T10:20:00Z |
| `limit` | string | 否 | 返回结果数量 \(默认：100，最大：500\) |
