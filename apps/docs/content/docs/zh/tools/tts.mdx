## 使用说明

使用来自 OpenAI、Deepgram、ElevenLabs、Cartesia、Google Cloud、Azure 和 PlayHT 的顶尖 AI 语音，将文本转换为自然流畅的语音。支持多种声音、语言及音频格式。

## 注意事项

- 类别：`tools`
- 类型：`tts`
