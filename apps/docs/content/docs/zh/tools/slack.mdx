<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">输入</div>

| 参数 | 类型 | 必填 | 描述 |
| --------- | ---- | -------- | ----------- |

| `destinationType` | string | 否 | 目标类型：channel 或 dm |
| `botToken` | string | 否 | 自定义机器人的 Bot Token |
| `channel` | string | 否 | 目标 Slack 频道 \(例如，#general\) |
| `dmUserId` | string | 否 | 私信目标 Slack 用户 |
| `text` | string | 是 | 要发送的消息文本 \(支持 Slack mrkdwn 格式\) |
| `thread_ts` | string | 否 | 要回复的消息时间戳 \(创建话题回复\) |
| `files` | file[] | 否 | 附加到消息的文件 |

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">输出</div>

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `message` | object | Slack 返回的包含所有属性的完整消息对象 |
| `ts` | string | 消息时间戳 |
| `channel` | string | 消息发送目标频道的 ID |

## 使用说明

将 Slack 集成到工作流中。支持发送消息、创建画布及读取消息。高级模式下需使用 Bot Token 而非 OAuth。可在触发模式下使用，当频道收到消息时自动触发工作流。
