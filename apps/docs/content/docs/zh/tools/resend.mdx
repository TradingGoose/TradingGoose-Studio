## 工具

### `resend_send`

使用您自己的 Resend API 密钥和发件人地址发送电子邮件

#### 输入

| 参数 | 类型 | 必填 | 描述 |
| --------- | ---- | -------- | ----------- |
| `fromAddress` | string | 是 | 发件人电子邮件地址 |
| `to` | string | 是 | 收件人电子邮件地址 |
| `subject` | string | 是 | 邮件主题 |
| `body` | string | 是 | 邮件正文内容 |
| `contentType` | string | 否 | 邮件正文的内容类型 \(text 或 html\) |
| `resendApiKey` | string | 是 | 用于发送邮件的 Resend API 密钥 |

#### 输出

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `success` | boolean | 邮件是否发送成功 |
| `to` | string | 收件人邮箱地址 |
| `subject` | string | 邮件主题 |

## 使用说明

将 Resend 集成到工作流中，可用于发送电子邮件，需要提供 API Key。
