## 工具

### `stagehand_agent`

运行自主网页智能体以完成任务并提取结构化数据

#### 输入

| 参数 | 类型 | 必填 | 描述 |
| --------- | ---- | -------- | ----------- |
| `startUrl` | string | 是 | 智能体启动的网页 URL |
| `task` | string | 是 | 在网站上需要完成的任务或实现的目标 |
| `variables` | json | 否 | 可选变量，用于替换任务中的占位符 \(格式：\{key: value\}\)。在任务中使用 %key% 引用 |
| `format` | string | 否 | 无描述 |
| `provider` | string | 否 | 要使用的 AI 提供商：openai 或 anthropic |
| `apiKey` | string | 是 | 所选提供商的 API 密钥 |
| `outputSchema` | json | 否 | 可选 JSON Schema，定义智能体应返回的数据结构 |

#### 输出

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `success` | boolean | 智能体任务是否成功完成 |
| `completed` | boolean | 任务是否已全部完成 |
| `message` | string | 状态消息或最终结果 |

## 使用说明

将 Stagehand Agent 集成到工作流中，可进行网页导航并执行任务。

## 注意事项

- 类别：`tools`
