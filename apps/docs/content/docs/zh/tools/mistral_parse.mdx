## 工具

### `mistral_parser`

使用 Mistral OCR API 解析 PDF 文档

#### 输入

| 参数 | 类型 | 必填 | 描述 |
| --------- | ---- | -------- | ----------- |
| `filePath` | string | 是 | 待处理 PDF 文档的 URL |
| `fileUpload` | object | 否 | 来自文件上传组件的文件上传数据 |
| `resultType` | string | 否 | 解析结果的类型 \(markdown、text 或 json\)，默认为 markdown |
| `includeImageBase64` | boolean | 否 | 在响应中包含 base64 编码的图像 |
| `pages` | array | 否 | 指定要处理的页面 \(页码数组，从 0 开始\) |
| `imageLimit` | number | 否 | 从 PDF 中提取图像的最大数量 |
| `imageMinSize` | number | 否 | 从 PDF 中提取图像的最小高度和宽度 |
| `apiKey` | string | 是 | Mistral API 密钥 \(MISTRAL_API_KEY\) |

#### 输出

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `success` | boolean | PDF 是否解析成功 |
| `content` | string | 以请求格式提取的内容 \(markdown、text 或 JSON\) |
| `jobId` | string | 唯一任务标识符 |
| `fileType` | string | 文件类型 \(例如：pdf\) |
| `fileName` | string | 原始文件名 |
| `source` | string | 来源类型 \(url\) |
| `pageCount` | number | 已处理的页数 |
| `model` | string | 所使用的 Mistral 模型 |
| `resultType` | string | 输出格式 \(markdown、text、json\) |
| `processedAt` | string | 处理时间戳 |
| `sourceUrl` | string | 来源 URL（如适用）|

## 使用说明

将 Mistral Parse 集成到工作流中。可从上传的 PDF 文档或 URL 中提取文本。

## 注意事项

- 类别：`tools`
- 类型：`mistral_parse`
