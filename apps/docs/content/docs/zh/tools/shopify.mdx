通过 ID 获取特定集合，包括其产品。使用此功能检索集合中的产品。

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">输入</div>

| 参数 | 类型 | 必填 | 描述 |
| --------- | ---- | -------- | ----------- |
| `shopDomain` | string | 是 | 您的 Shopify 店铺域名 \(例如，mystore.myshopify.com\) |
| `collectionId` | string | 是 | 集合 ID \(例如，gid://shopify/Collection/123456789\) |
| `productsFirst` | number | 否 | 从该集合返回的产品数量 \(默认值：50，最大值：250\) |

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">输出</div>

请参阅此操作的块输出。

</div>

## 使用说明

将 Shopify 集成到您的工作流程中。管理产品、订单、客户和库存。创建、读取、更新和删除产品。列出并管理订单。处理客户数据并调整库存水平。
