## 工具

### `file_parser`

解析一个或多个上传的文件或来自 URL 的文件（文本、PDF、CSV、图像等）

#### 输入

| 参数 | 类型 | 必填 | 描述 |
| --------- | ---- | -------- | ----------- |
| `filePath` | string | 是 | 文件路径。可以是单个路径、URL 或路径数组。 |
| `fileType` | string | 否 | 要解析的文件类型（如未指定则自动检测） |

#### 输出

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `files` | array | 包含内容和元数据的已解析文件数组 |
| `combinedContent` | string | 所有已解析文件的合并内容 |

## 使用说明

将 File 集成到工作流中。可手动上传文件或插入文件 URL。

## 注意事项

- 类别：`tools`
- 类型：`file`
