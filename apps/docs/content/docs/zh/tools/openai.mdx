## 工具

### `openai_embeddings`

使用 OpenAI 从文本生成嵌入向量

#### 输入

| 参数 | 类型 | 是否必填 | 描述 |
| --------- | ---- | -------- | ----------- |
| `input` | string | 是 | 用于生成嵌入向量的文本 |
| `model` | string | 否 | 用于生成嵌入向量的模型 |
| `encodingFormat` | string | 否 | 嵌入向量的返回格式 |
| `apiKey` | string | 是 | OpenAI API 密钥 |

#### 输出

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `success` | boolean | 操作成功状态 |
| `embeddings` | array | 嵌入向量数组 |
| `model` | string | 用于生成嵌入向量的模型 |
| `prompt_tokens` | number | 提示词中的 Token 数量 |

## 使用说明

将嵌入向量集成到工作流中，可从文本生成嵌入向量。

## 注意事项

- 类别：`tools`
- 类型：`openai`
