在远程 SSH 服务器上执行 Shell 命令

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">输入</div>

| 参数 | 类型 | 必填 | 描述 |
| --------- | ---- | -------- | ----------- |
| `host` | string | 是 | SSH 服务器主机名或 IP 地址 |
| `port` | number | 是 | SSH 服务器端口 \(默认：22\) |
| `username` | string | 是 | SSH 用户名 |
| `password` | string | 否 | 身份验证密码 \(不使用私钥时\) |
| `privateKey` | string | 否 | 身份验证私钥 \(OpenSSH 格式\) |
| `passphrase` | string | 否 | 加密私钥的密码短语 |
| `command` | string | 是 | 在远程服务器上执行的 Shell 命令 |
| `workingDirectory` | string | 否 | 命令执行的工作目录 |

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">输出</div>

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `stdout` | string | 命令的标准输出 |
| `stderr` | string | 标准错误输出 |
| `exitCode` | number | 命令退出码 |
| `success` | boolean | 命令是否执行成功 \(退出码为 0\) |
