## 工具

### `duckduckgo_search`

使用 DuckDuckGo 即时答案 API 搜索网络。返回查询的即时答案、摘要和相关主题。无需 API 密钥，免费使用。

#### 输入

| 参数 | 类型 | 是否必填 | 描述 |
| --------- | ---- | -------- | ----------- |
| `query` | string | 是 | 要执行的搜索查询 |
| `noHtml` | boolean | 否 | 从结果文本中移除 HTML \(默认值：true\) |
| `skipDisambig` | boolean | 否 | 跳过消歧义结果 \(默认值：false\) |

#### 输出

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `heading` | string | 即时答案的标题 |
| `abstract` | string | 主题的简短摘要 |
| `abstractText` | string | 摘要的纯文本版本 |
| `abstractSource` | string | 摘要来源 \(例如：Wikipedia\) |
| `abstractURL` | string | 摘要来源的 URL |
| `image` | string | 与主题相关的图片 URL |
| `answer` | string | 直接答案（如有）\(例如：用于计算结果\) |
| `answerType` | string | 答案类型 \(例如：calc、ip 等\) |
| `type` | string | 响应类型：A \(文章\)、D \(消歧义\)、C \(分类\)、N \(名称\)、E \(独占\) |
| `FirstURL` | string | 相关主题的 URL |
| `Text` | string | 相关主题的描述 |

## 使用说明

使用 DuckDuckGo 即时答案 API 搜索网络。返回即时答案、摘要、相关主题等内容。无需 API 密钥，免费使用。
