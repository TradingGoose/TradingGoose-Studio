## 工具

### `linkup_search`

使用 Linkup 搜索网络信息

#### 输入

| 参数 | 类型 | 必填 | 描述 |
| --------- | ---- | -------- | ----------- |
| `q` | string | 是 | 搜索查询词 |
| `depth` | string | 是 | 搜索深度 \(必须为 "standard" 或 "deep"\) |
| `outputType` | string | 是 | 返回结果的输出类型 \(必须为 "sourcedAnswer" 或 "searchResults"\) |
| `apiKey` | string | 是 | 请输入您的 Linkup API 密钥 |

#### 输出

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `answer` | string | 搜索查询的来源答案 |

## 使用说明

将 Linkup 集成到工作流中，支持网络搜索。

## 注意事项

- 类别：`tools`
- 类型：`linkup`
