## 工具

### `serper_search`

一款强大的网络搜索工具，通过 Serper.dev API 提供对 Google 搜索结果的访问。支持多种搜索类型，包括常规网页搜索、新闻、地点和图片，每条结果均包含相关元数据，如标题、URL、摘要及特定类型的附加信息。

#### 输入

| 参数 | 类型 | 必填 | 描述 |
| --------- | ---- | -------- | ----------- |
| `query` | string | 是 | 搜索查询词 |
| `num` | number | 否 | 返回结果数量 |
| `gl` | string | 否 | 搜索结果的国家代码 |
| `hl` | string | 否 | 搜索结果的语言代码 |
| `type` | string | 否 | 搜索类型 |
| `apiKey` | string | 是 | Serper API 密钥 |

#### 输出

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `searchResults` | json | 搜索结果数据 |

## 使用说明

将 Serper 集成至工作流中，可用于网络搜索。
