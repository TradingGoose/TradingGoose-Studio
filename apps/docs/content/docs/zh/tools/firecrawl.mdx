<div className="text-base font-semibold mb-2">抓取 (<code>firecrawl_scrape</code>)</div>

从网页中提取结构化内容，支持全面的元数据。可将内容转换为 Markdown 或 HTML 格式，同时捕获 SEO 元数据、Open Graph 标签及页面信息。

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">输入</div>

| 参数 | 类型 | 必填 | 描述 |
| --------- | ---- | -------- | ----------- |
| `url` | string | 是 | 要抓取内容的 URL |
| `scrapeOptions` | json | 否 | 内容抓取选项 |
| `apiKey` | string | 是 | Firecrawl API 密钥 |

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">输出</div>

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `markdown` | string | Markdown 格式的页面内容 |
| `html` | string | 页面原始 HTML 内容 |

</div>
