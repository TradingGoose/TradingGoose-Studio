<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">输入</div>

| 参数 | 类型 | 必填 | 描述 |
| --------- | ---- | -------- | ----------- |
| `path` | string | 是 | 文件在 Dropbox 中的保存路径 \(例如：/folder/document.pdf\) |
| `fileContent` | string | 是 | 要上传文件的 base64 编码内容 |
| `fileName` | string | 否 | 可选文件名 \(当路径为文件夹时使用\) |
| `mode` | string | 否 | 写入模式：add \(默认\) 或 overwrite |
| `autorename` | boolean | 否 | 若为 true，发生冲突时自动重命名文件 |
| `mute` | boolean | 否 | 若为 true，不向用户发送此次上传的通知 |

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">输出</div>

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `id` | string | 文件的唯一标识符 |
| `name` | string | 文件名称 |
| `path_display` | string | 文件的显示路径 |
| `path_lower` | string | 文件的小写路径 |
| `size` | number | 文件大小（字节） |
| `client_modified` | string | 客户端修改时间 |
| `server_modified` | string | 服务器修改时间 |
| `rev` | string | 版本标识符 |
