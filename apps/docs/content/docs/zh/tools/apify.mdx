| `actorId` | string | 是 | Actor ID 或 用户名/actor名称 \(例如："janedoe/my-actor" 或 actor ID\) |
| `input` | string | 否 | JSON 字符串格式的 Actor 输入。请参阅 actor 文档了解必填字段。 |
| `timeout` | number | 否 | 超时时间（秒）\(默认：actor 默认值\) |
| `build` | string | 否 | 要运行的 Actor 构建版本 \(例如："latest"、"beta" 或构建标签/编号\) |

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">输出</div>

| 参数 | 类型 | 描述 |

| `success` | boolean | Actor 运行是否成功 |
| `runId` | string | APIFY 运行 ID |
| `status` | string | 运行状态 \(SUCCEEDED、FAILED 等\) |

</div>

## 使用说明

将 Apify 集成到您的工作流中。使用自定义输入运行任意 Apify Actor 并获取结果。支持同步与异步两种执行模式，并自动获取数据集。
