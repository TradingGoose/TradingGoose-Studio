## 工具

### `openai_chat`

#### 输入

| 参数 | 类型 | 必填 | 描述 |
| --------- | ---- | -------- | ----------- |

#### 输出

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `content` | string | 翻译后的文本 |
| `model` | string | 使用的模型 |
| `tokens` | json | Token 用量 |
