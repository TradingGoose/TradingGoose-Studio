<div className="text-base font-semibold mb-2">发起通话 (<code>twilio_voice_make_call</code>)</div>

使用 Twilio Voice API 发起一个出站电话呼叫。

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">输入</div>

| 参数 | 类型 | 必填 | 描述 |
| --------- | ---- | -------- | ----------- |
| `to` | string | 是 | 被叫电话号码 \(E.164 格式，例如 +14155551234\) |
| `from` | string | 是 | 您的 Twilio 主叫电话号码 \(E.164 格式\) |
| `url` | string | 否 | 返回通话 TwiML 指令的 URL |
| `twiml` | string | 否 | 要执行的 TwiML 指令（URL 的替代方案）。请使用方括号代替尖括号，例如 \[Response\]\[Say\]Hello\[/Say\]\[/Response\] |
| `statusCallback` | string | 否 | 通话状态更新的 Webhook URL |
| `statusCallbackMethod` | string | 否 | 状态回调的 HTTP 方法 \(GET 或 POST\) |
| `accountSid` | string | 是 | Twilio Account SID |
| `authToken` | string | 是 | Twilio Auth Token |
| `record` | boolean | 否 | 是否录制通话 |
| `recordingStatusCallback` | string | 否 | 录音状态更新的 Webhook URL |
| `timeout` | number | 否 | 放弃前等待接听的时间 \(秒，默认值：60\) |
| `machineDetection` | string | 否 | 答录机检测：Enable 或 DetectMessageEnd |

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">输出</div>

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `success` | boolean | 通话是否成功发起 |
| `callSid` | string | 通话的唯一标识符 |
| `status` | string | 通话状态 \(queued、ringing、in-progress、completed 等\) |
| `direction` | string | 通话方向 \(outbound-api\) |
| `from` | string | 主叫电话号码 |
| `to` | string | 被叫电话号码 |
| `duration` | number | 通话时长（秒） |
| `price` | string | 通话费用 |
| `priceUnit` | string | 费用币种 |

</div>

| `duration` | number | 录音时长（秒） |

| `channels` | number | 声道数 \(1 为单声道，2 为双声道\) |

| `priceUnit` | string | 费用币种 |
