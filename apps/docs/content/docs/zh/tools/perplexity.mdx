## 工具

### `perplexity_chat`

使用 Perplexity AI 对话模型生成补全内容

#### 输入

| 参数 | 类型 | 必填 | 描述 |
| --------- | ---- | -------- | ----------- |
| `systemPrompt` | string | 否 | 用于引导模型行为的系统提示词 |
| `content` | string | 是 | 发送给模型的用户消息内容 |
| `model` | string | 是 | 用于对话补全的模型（例如：sonar、mistral） |
| `max_tokens` | number | 否 | 最大生成 Token 数量 |
| `temperature` | number | 否 | 采样温度，范围 0 至 1 |
| `apiKey` | string | 是 | Perplexity API 密钥 |

#### 输出

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `success` | boolean | 操作成功状态 |
| `content` | string | 生成的文本内容 |
| `model` | string | 用于生成的模型 |
| `prompt_tokens` | number | 提示词中的 token 数量 |
| `completion_tokens` | number | 补全内容中的 token 数量 |

## 使用说明

将 Perplexity 集成到工作流中。可使用 Perplexity AI 对话模型生成补全内容。

- 类别：`tools`
