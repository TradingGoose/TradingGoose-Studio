## 工具

### `trading_get_holdings`

#### 输入

| 参数 | 类型 | 必填 | 描述 |
| --------- | ---- | -------- | ----------- |

#### 输出

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `summary` | string | 持仓检索状态 |
| `provider` | string | 所使用的提供商 |
| `holdings` | json | 统一账户快照数据 |

## 使用说明

统一持仓模块，返回 Alpaca、Tradier 或 Robinhood 的账户快照。

## 注意事项

- 类别：`tools`
- 类型：`trading_holdings`
