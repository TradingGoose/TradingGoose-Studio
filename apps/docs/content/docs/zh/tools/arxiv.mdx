| `searchField` | string | 否 | 搜索字段：all（全部）、ti \(title\)（标题）、au \(author\)（作者）、abs \(abstract\)（摘要）、co \(comment\)（评论）、jr \(journal\)（期刊）、cat \(category\)（分类）、rn \(report number\)（报告编号） |
| `maxResults` | number | 否 | 返回结果的最大数量 \(默认值：10，最大值：2000\) |
| `sortBy` | string | 否 | 排序依据：relevance（相关性）、lastUpdatedDate（最后更新日期）、submittedDate（提交日期）\(默认值：relevance\) |
| `sortOrder` | string | 否 | 排序顺序：ascending（升序）、descending（降序）\(默认值：descending\) |

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">输出</div>
