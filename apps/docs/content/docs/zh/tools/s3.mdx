## 使用说明

将 S3 集成到工作流中。支持上传文件、下载对象、列出存储桶内容、删除对象以及在存储桶之间复制对象。需要提供 AWS 访问密钥和秘密访问密钥。

## 注意事项

- 类别: `tools`
- 类型: `s3`
