| `topic` | string | 是 | 会议主题 |
| `type` | number | 否 | 会议类型：1=即时会议，2=定时会议，3=循环会议（无固定时间），8=循环会议（固定时间） |
| `startTime` | string | 否 | ISO 8601 格式的会议开始时间 \(例如：2025-06-03T10:00:00Z\) |
| `duration` | number | 否 | 会议时长（分钟） |
| `timezone` | string | 否 | 会议时区 \(例如：America/Los_Angeles\) |
| `password` | string | 否 | 会议密码 |
| `agenda` | string | 否 | 会议议程 |
| `hostVideo` | boolean | 否 | 开启主持人视频入会 |
| `participantVideo` | boolean | 否 | 开启参与者视频入会 |
| `joinBeforeHost` | boolean | 否 | 允许参与者在主持人前加入 |
| `muteUponEntry` | boolean | 否 | 参与者入会时静音 |
| `waitingRoom` | boolean | 否 | 启用等候室 |
| `autoRecording` | string | 否 | 自动录制设置：local（本地）、cloud（云端）或 none（不录制） |

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">输出</div>

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `id` | number | 会议 ID |
| `uuid` | string | 会议 UUID |
| `host_id` | string | 主持人用户 ID |
| `host_email` | string | 主持人电子邮件 |
| `topic` | string | 会议主题 |
| `type` | number | 会议类型 |
| `status` | string | 会议状态 |
| `start_time` | string | 开始时间 |
| `duration` | number | 时长（分钟） |
| `timezone` | string | 时区 |
| `agenda` | string | 会议议程 |
| `created_at` | string | 创建时间戳 |
| `start_url` | string | 主持人开始链接 |
| `join_url` | string | 参与者加入链接 |
| `password` | string | 会议密码 |
