<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">输入</div>

| 参数 | 类型 | 必填 | 描述 |

| `sessionId` | string | 否 | 用于保持对话连续性的会话 ID |
| `genius` | boolean | 否 | 启用天才模式以进行更深入的分析 \(速度较慢但更准确\) |
| `apiKey` | string | 是 | Greptile API 密钥 |
| `githubToken` | string | 是 | 具有仓库读取权限的 GitHub 个人访问令牌 |

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">输出</div>

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `message` | string | AI 生成的查询答案 |
| `repository` | string | 仓库名称 \(owner/repo\) |
| `remote` | string | Git 远程端 \(github/gitlab\) |
| `branch` | string | 分支名称 |
| `filepath` | string | 文件路径 |
| `linestart` | number | 起始行号 |
| `lineend` | number | 结束行号 |
| `summary` | string | 代码段摘要 |

<div className="text-base font-semibold mb-2">索引仓库 (<code>greptile_index_repo</code>)</div>

提交一个仓库以供 Greptile 建立索引。索引完成后，才能对该仓库进行查询。小型仓库需要 3-5 分钟，较大的仓库可能需要超过一小时。

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">输入</div>

| 参数 | 类型 | 必填 | 描述 |
| --------- | ---- | -------- | ----------- |
| `remote` | string | 是 | Git 远程类型：github 或 gitlab |
| `repository` | string | 是 | 仓库格式为 owner/repo（例如："facebook/react"） |
| `branch` | string | 是 | 要索引的分支（例如："main" 或 "master"） |
| `reload` | boolean | 否 | 强制重新索引，即使已经完成索引 |
| `notify` | boolean | 否 | 索引完成后发送电子邮件通知 |
| `apiKey` | string | 是 | Greptile API 密钥 |
| `githubToken` | string | 是 | 具有仓库读取权限的 GitHub 个人访问令牌 |

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">输出</div>

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `repositoryId` | string | 已索引仓库的唯一标识符（格式：remote:branch:owner/repo） |
| `statusEndpoint` | string | 用于检查索引状态的 URL 端点 |

</div>

## 使用说明

使用 Greptile 通过自然语言查询和搜索代码库。获取关于代码的 AI 生成解答，查找相关文件，并深入理解复杂代码库。
