在 Intercom 中通过电子邮件、external_id 或角色创建新联系人

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">输入</div>

| 参数 | 类型 | 必填 | 描述 |
| --------- | ---- | -------- | ----------- |
| `role` | string | 否 | 联系人的角色。接受 'user' 或 'lead'。若未指定，默认为 'lead'。 |
| `email` | string | 否 | 联系人的电子邮件地址 |
| `external_id` | string | 否 | 由客户端提供的联系人唯一标识符 |
| `phone` | string | 否 | 联系人的电话号码 |
| `name` | string | 否 | 联系人的姓名 |
| `avatar` | string | 否 | 联系人的头像图片 URL |
| `signed_up_at` | number | 否 | 用户注册时间，以 Unix 时间戳表示 |
| `last_seen_at` | number | 否 | 用户最后活跃时间，以 Unix 时间戳表示 |
| `owner_id` | string | 否 | 已被分配该联系人账户所有权的管理员 ID |
| `unsubscribed_from_emails` | boolean | 否 | 联系人是否已取消订阅电子邮件 |
| `custom_attributes` | string | 否 | JSON 对象格式的自定义属性 \(例如，\{"attribute_name": "value"\}\) |
| `company_id` | string | 否 | 创建联系人时关联的公司 ID |

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">输出</div>

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `id` | string | 联系人的唯一标识符 |
| `type` | string | 列表类型 |
| `role` | string | 联系人角色（user 或 lead） |
| `email` | string | 联系人的电子邮件地址 |
| `phone` | string | 联系人的电话号码 |
| `name` | string | 联系人的姓名 |
| `avatar` | string | 联系人的头像 URL |
| `owner_id` | string | 分配给该联系人的管理员 ID |
| `external_id` | string | 联系人的外部标识符 |
| `created_at` | number | 联系人创建时间的 Unix 时间戳 |
| `updated_at` | number | 联系人最后更新时间的 Unix 时间戳 |
