## 使用说明

通过 Data API 将 Amazon RDS Aurora Serverless 集成到工作流中。无需管理数据库连接，即可执行查询、插入、更新、删除及原始 SQL 操作。

## 注意事项

- 分类：`tools`
- 类型：`rds`
