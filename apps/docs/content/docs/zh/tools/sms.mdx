## 工具

### `sms_send`

使用由 Twilio 驱动的内部短信服务发送短信消息

#### 输入

| 参数 | 类型 | 必填 | 描述 |
| --------- | ---- | -------- | ----------- |
| `to` | string | 是 | 收件人电话号码 \(包含国家代码，例如 +1234567890\) |
| `body` | string | 是 | 短信消息内容 |

#### 输出

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `success` | boolean | 短信是否发送成功 |
| `to` | string | 收件人电话号码 |

## 使用说明

直接使用由 Twilio 驱动的内部短信服务发送短信消息。无需外部配置或 OAuth 授权。非常适合在工作流中发送通知、告警或通用文本消息。需要包含国家代码的有效电话号码。

## 注意事项

- 类别：`tools`
- 类型：`sms`
