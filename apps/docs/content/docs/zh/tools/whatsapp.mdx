## 工具

### `whatsapp_send_message`

发送 WhatsApp 消息

#### 输入

| 参数 | 类型 | 必填 | 描述 |
| --------- | ---- | -------- | ----------- |
| `phoneNumber` | string | 是 | 收件人电话号码（含国家代码）|
| `message` | string | 是 | 要发送的消息内容 |
| `phoneNumberId` | string | 是 | WhatsApp Business 电话号码 ID |

#### 输出

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `success` | boolean | WhatsApp 消息发送成功状态 |
| `messageId` | string | WhatsApp 消息唯一标识符 |
| `phoneNumber` | string | 收件人电话号码 |
| `status` | string | 消息投递状态 |

## 使用说明

将 WhatsApp 集成到工作流中，可发送消息。

## 注意事项

- 类别：`tools`
- 类型：`whatsapp`
