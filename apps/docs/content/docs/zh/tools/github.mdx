## 使用说明

将 GitHub 集成到工作流中。可获取 PR 详情、创建 PR 评论、获取仓库信息及获取最新提交。可在触发模式下使用，当 PR 被创建、被评论或有提交推送时触发工作流。

## 注意事项

- 类别：`tools`
- 类型：`github`
