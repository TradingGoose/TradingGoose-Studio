上传文件至远程 SFTP 服务器

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">输入</div>

| 参数 | 类型 | 必填 | 描述 |
| --------- | ---- | -------- | ----------- |
| `host` | string | 是 | SFTP 服务器主机名或 IP 地址 |
| `port` | number | 是 | SFTP 服务器端口 \(默认：22\) |
| `username` | string | 是 | SFTP 用户名 |
| `password` | string | 否 | 身份验证密码 \(不使用私钥时\) |
| `privateKey` | string | 否 | 身份验证私钥 \(OpenSSH 格式\) |
| `passphrase` | string | 否 | 加密私钥的密码短语 |
| `remotePath` | string | 是 | 远程服务器上的目标目录 |
| `files` | file[] | 否 | 要上传的文件 |
| `fileContent` | string | 否 | 直接上传的文件内容 \(适用于文本文件\) |
| `fileName` | string | 否 | 使用直接内容时的文件名 |
| `overwrite` | boolean | 否 | 是否覆盖已有文件 \(默认：true\) |
| `permissions` | string | 否 | 文件权限 \(例如：0644\) |

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">输出</div>

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `success` | boolean | 上传是否成功 |
| `uploadedFiles` | json | 已上传文件详情数组 \(文件名、远程路径、大小\) |

</div>
