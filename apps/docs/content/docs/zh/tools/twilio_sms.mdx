## 工具

### `twilio_send_sms`

使用 Twilio API 向单个或多个收件人发送文本消息。

#### 输入

| 参数 | 类型 | 必填 | 描述 |
| --------- | ---- | -------- | ----------- |
| `phoneNumbers` | string | 是 | 接收消息的电话号码，以换行符分隔 |
| `message` | string | 是 | 要发送的消息内容 |
| `accountSid` | string | 是 | Twilio 账户 SID |
| `authToken` | string | 是 | Twilio 身份验证令牌 |
| `fromNumber` | string | 是 | 用于发送消息的 Twilio 电话号码 |

#### 输出

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `success` | boolean | 短信发送成功状态 |
| `messageId` | string | Twilio 唯一消息标识符 \(SID\) |
| `status` | string | 来自 Twilio 的消息投递状态 |
| `fromNumber` | string | 消息发送方的电话号码 |

## 使用说明

将 Twilio 集成到工作流中，可发送短信消息。
