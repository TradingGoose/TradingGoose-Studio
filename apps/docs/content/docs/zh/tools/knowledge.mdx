使用向量相似性在知识库中搜索相似内容

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">输入</div>

| 参数 | 类型 | 必填 | 描述 |
| --------- | ---- | -------- | ----------- |
| `knowledgeBaseId` | string | 是 | 要搜索的知识库 ID |
| `query` | string | 否 | 搜索查询文本 \(使用标签过滤器时可选\) |
| `topK` | number | 否 | 返回的最相似结果数量 \(1-100\) |
| `tagFilters` | array | 否 | 包含 tagName 和 tagValue 属性的标签过滤器数组 |
| `items` | object | 否 | 无描述 |
| `properties` | string | 否 | 无描述 |
| `tagName` | string | 否 | 无描述 |
| `tagValue` | string | 否 | 无描述 |
| `tagFilters` | string | 否 | 无描述 |

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">输出</div>

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `documentId` | string | 文档 ID |
| `documentName` | string | 文档名称 |
| `content` | string | 结果内容 |
| `chunkIndex` | number | 该块在文档中的索引 |
| `similarity` | number | 结果的相似度得分 |

</div>

将知识库集成到工作流中。可搜索、上传数据块及创建文档。
