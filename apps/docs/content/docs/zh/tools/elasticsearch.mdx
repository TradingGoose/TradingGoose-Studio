<div className="text-base font-semibold mb-2">搜索（<code>elasticsearch_search</code>）</div>

使用 Query DSL 在 Elasticsearch 中搜索文档。返回匹配的文档及其评分和元数据。

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">输入</div>

| 参数 | 类型 | 是否必填 | 描述 |
| --------- | ---- | -------- | ----------- |
| `deploymentType` | string | 是 | 部署类型：self_hosted 或 cloud |
| `host` | string | 否 | Elasticsearch 主机 URL（适用于自托管） |
| `cloudId` | string | 否 | Elastic Cloud ID（适用于云部署） |
| `authMethod` | string | 是 | 认证方式：api_key 或 basic_auth |
| `apiKey` | string | 否 | Elasticsearch API 密钥 |
| `username` | string | 否 | 基本认证用户名 |
| `password` | string | 否 | 基本认证密码 |
| `index` | string | 是 | 要搜索的索引名称 |
| `query` | string | 否 | JSON 字符串格式的 Query DSL |
| `from` | number | 否 | 分页起始偏移量（默认：0） |
| `size` | number | 否 | 返回结果数量（默认：10） |
| `sort` | string | 否 | JSON 字符串格式的排序规范 |
| `sourceIncludes` | string | 否 | 以逗号分隔的字段列表，用于指定 _source 中包含的字段 |
| `sourceExcludes` | string | 否 | 以逗号分隔的字段列表，用于指定从 _source 中排除的字段 |
| `trackTotalHits` | boolean | 否 | 是否精确统计命中总数（默认：true） |

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">输出</div>

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `took` | number | 搜索所用时间（毫秒） |
| `timed_out` | boolean | 搜索是否超时 |
| `hits` | object | 搜索结果，包含总数和匹配文档 |

</div>
