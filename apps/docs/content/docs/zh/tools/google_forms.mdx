## 工具

### `google_forms_get_responses`

#### 输入

| 参数 | 类型 | 必填 | 描述 |
| --------- | ---- | -------- | ----------- |

#### 输出

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `data` | json | 响应或响应列表 |

## 使用说明

将 Google Forms 集成到您的工作流程中。提供表单 ID 以列出响应，或指定响应 ID 以获取单个响应。需要 OAuth 授权。

## 注意事项

- 类别：`tools`
- 类型：`google_forms`
