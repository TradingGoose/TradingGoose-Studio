更新 Jira 问题

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">输入</div>

| 参数 | 类型 | 必填 | 描述 |
| --------- | ---- | -------- | ----------- |
| `domain` | string | 是 | 您的 Jira 域名 \(例如：yourcompany.atlassian.net\) |
| `projectId` | string | 否 | 用于更新问题的 Jira 项目 ID。若未提供，则检索所有问题。 |
| `issueKey` | string | 是 | 要更新的 Jira 问题键 |
| `summary` | string | 否 | 问题的新摘要 |
| `description` | string | 否 | 问题的新描述 |
| `status` | string | 否 | 问题的新状态 |
| `priority` | string | 否 | 问题的新优先级 |
| `assignee` | string | 否 | 问题的新负责人 |
| `cloudId` | string | 否 | 实例的 Jira Cloud ID。若未提供，将通过域名自动获取。 |

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">输出</div>

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `ts` | string | 操作时间戳 |
| `issueKey` | string | 更新后的议题键值 \(例如，PROJ-123\) |

</div>

创建一个 Jira 议题

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">输入</div>

| 参数 | 类型 | 必填 | 描述 |
| --------- | ---- | -------- | ----------- |
| `domain` | string | 是 | 您的 Jira 域名 \(例如：yourcompany.atlassian.net\) |
| `projectId` | string | 是 | 问题的项目 ID |
| `summary` | string | 是 | 问题的摘要 |
| `description` | string | 否 | 问题的描述 |
| `priority` | string | 否 | 问题的优先级 ID 或名称 \(例如："10000" 或 "High"\) |
| `assignee` | string | 否 | 问题的经办人账户 ID |
| `cloudId` | string | 否 | 实例的 Jira Cloud ID。若未提供，将通过域名自动获取。|
| `issueType` | string | 是 | 要创建的问题类型 \(例如：Task、Story\) |
| `labels` | array | 否 | 问题的标签 \(标签名称数组\) |
| `duedate` | string | 否 | 问题的截止日期 \(格式：YYYY-MM-DD\) |
| `reporter` | string | 否 | 问题的报告人账户 ID |
| `environment` | string | 否 | 问题的环境信息 |
| `customFieldId` | string | 否 | 自定义字段 ID \(例如：customfield_10001\) |
| `customFieldValue` | string | 否 | 自定义字段的值 |

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">输出</div>

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `ts` | string | 操作的时间戳 |
| `issueKey` | string | 已创建的问题键值 \(例如：PROJ-123\) |
| `summary` | string | 问题摘要 |
| `url` | string | 已创建问题的 URL |

</div>
