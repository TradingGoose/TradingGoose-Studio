<div className="text-base font-semibold mb-2">反向链接 (<code>ahrefs_backlinks</code>)</div>

获取指向目标域名或URL的反向链接列表。返回每条反向链接的详细信息，包括来源URL、锚文本和域名评分。

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">输入</div>

| 参数 | 类型 | 必填 | 描述 |
| --------- | ---- | -------- | ----------- |
| `target` | string | 是 | 要分析的目标域名或URL |
| `mode` | string | 否 | 分析模式：domain（整个域名）、prefix（URL前缀）、subdomains（包含所有子域名）、exact（精确URL匹配） |
| `date` | string | 否 | 历史数据日期，格式为YYYY-MM-DD（默认为今天） |
| `limit` | number | 否 | 返回结果的最大数量（默认：100） |
| `offset` | number | 否 | 分页时跳过的结果数量 |
| `apiKey` | string | 是 | Ahrefs API密钥 |

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">输出</div>

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `urlFrom` | string | 包含该反向链接的页面URL |
| `urlTo` | string | 被链接的目标URL |
| `anchor` | string | 链接的锚文本 |
| `domainRatingSource` | number | 链接域名的域名评分 |
| `isDofollow` | boolean | 该链接是否为dofollow链接 |
| `firstSeen` | string | 反向链接首次被发现的时间 |

</div>

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">输入参数</div>

| `date` | string | 否 | 历史数据的日期，格式为 YYYY-MM-DD\（默认为今天\） |
| `apiKey` | string | 是 | Ahrefs API 密钥 |

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">输出</div>

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `total` | number | 活跃外链总数 |
| `dofollow` | number | dofollow 外链数量 |
| `nofollow` | number | nofollow 外链数量 |
| `text` | number | 文字外链数量 |
| `image` | number | 图片外链数量 |
