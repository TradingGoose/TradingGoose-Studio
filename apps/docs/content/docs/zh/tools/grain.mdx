| `participantScope` | string | 否 | 筛选："internal"（内部）或 "external"（外部） |

| `teamId` | string | 否 | 按团队 UUID 筛选 |
| `meetingTypeId` | string | 否 | 按会议类型 UUID 筛选 |
| `includeHighlights` | boolean | 否 | 在响应中包含亮点/片段 |
| `includeParticipants` | boolean | 否 | 在响应中包含参与者列表 |
| `includeAiSummary` | boolean | 否 | 包含 AI 生成的摘要 |

| `end_datetime` | string | ISO8601 结束时间戳 |
| `duration_ms` | number | 时长（毫秒） |
| `media_type` | string | audio（音频）、transcript（转录）或 video（视频） |
| `source` | string | 录音来源 |
| `url` | string | 在 Grain 中查看的 URL |
| `thumbnail_url` | string | 缩略图 URL |
| `tags` | array | 标签数组 |
| `teams` | array | 录音所属的团队 |
