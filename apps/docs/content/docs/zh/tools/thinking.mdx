## 工具

### `thinking_tool`

处理提供的思路/指令，使其可用于后续步骤。

#### 输入

| 参数 | 类型 | 必填 | 描述 |
| --------- | ---- | -------- | ----------- |
| `thought` | string | 是 | 您的内部推理、分析或思维过程。在回复前，使用此参数逐步思考问题。 |

#### 输出

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `acknowledgedThought` | string | 已确认的思维过程 |

## 使用说明

添加一个步骤，让模型在继续之前明确概述其思维过程。这可以通过鼓励逐步分析来提升推理质量。

## 注意事项

- 类别：`tools`
- 类型：`thinking`
