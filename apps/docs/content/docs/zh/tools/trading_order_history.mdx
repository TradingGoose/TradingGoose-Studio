## 工具

### `trading_order_history`

检索指定日期时间范围内工作流的订单提交记录。

#### 输入

| 参数 | 类型 | 必填 | 描述 |
| --------- | ---- | -------- | ----------- |
| `startDate` | string | 是 | 开始日期时间 \(ISO 8601\)。 |
| `endDate` | string | 是 | 结束日期时间 \(ISO 8601\)。 |
| `workflowId` | string | 否 | 可选的工作流 ID，用于筛选。默认为当前工作流执行上下文。 |

#### 输出

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `id` | string | 订单历史记录 ID |
| `provider` | string | 交易提供商 |
| `recordedAt` | string | 记录时间戳 |
| `workflowId` | string | 工作流 ID |
| `workflowExecutionId` | string | 工作流执行 ID |

## 使用说明

获取交易操作工具在所选工作流和日期范围内记录的订单提交历史记录。
