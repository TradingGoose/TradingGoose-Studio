## 工具

### `trading_place_order`

#### 输入

| 参数 | 类型 | 必填 | 描述 |
| --------- | ---- | -------- | ----------- |

#### 输出

| 参数 | 类型 | 描述 |
| --------- | ---- | ----------- |
| `summary` | string | 订单提交状态 |
| `provider` | string | 所用提供商 |
| `order` | json | 订单载荷及原始响应 |

## 使用说明

统一交易操作模块，支持多家券商，兼容 OAuth 及 API 密钥两种身份验证方式。

## 注意事项

- 类别：`tools`
- 类型：`trading_action`
