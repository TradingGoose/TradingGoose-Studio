ワークフローエディタは、TradingGooseでAIワークフローを構築するための中核となる視覚的インターフェースです。ブロックをドラッグ、ドロップ、接続して強力な自動化パイプラインを作成できるキャンバスを提供します。

## 概要

| プロパティ | 値 |
|----------|-------|
| **キー** | `editor_workflow` |
| **カテゴリ** | エディタ |
| **アイコン** | ワークフロー |

## 機能

- **ビジュアルキャンバス**: ワークフロー設計のための直感的なドラッグ＆ドロップインターフェース
- **ブロック配置**: ブロックライブラリからキャンバスにブロックを追加
- **接続管理**: ブロックの出力と入力間の接続を描画
- **ズーム＆パン**: ズームとパンコントロールで大規模なワークフローをナビゲート
- **ブロック設定**: ブロックをクリックして設定を構成
- **リアルタイム検証**: 接続の互換性に関する即時フィードバック

## 使用方法

ワークフローエディタは通常、ワークスペースの主要なウィジェットであり、AIワークフローを設計および修正するための視覚的環境を提供します。

### 基本操作

1. **ブロックの追加**: サイドバーからブロックをドラッグするか、コンテキストメニューを使用
2. **ブロックの接続**: 出力ポートから入力ポートへクリック＆ドラッグ
3. **ブロックの設定**: ブロックをクリックして設定パネルを開く
4. **要素の削除**: 選択して削除キーを押すか、コンテキストメニューを使用
5. **元に戻す/やり直し**: Ctrl+Z / Ctrl+Shift+Zで履歴ナビゲーション

<Callout>
このウィジェットは、コンソール、チャット、変数などの他のワークフローウィジェットとの同期のための**pairColor**メカニズムをサポートしています。
</Callout>
