Workflow Chat ウィジェットは、ワークフローブロックとやり取りするためのインタラクティブなチャットインターフェースを提供します。AI ワークフローとの会話型インタラクションを可能にします。

## 概要

| プロパティ | 値 |
|-----------|----|
| **キー** | `workflow_chat` |
| **カテゴリ** | ユーティリティ |
| **アイコン** | MessageCircle |

## 特徴

- **インタラクティブなチャット**: ワークフローにメッセージを送り、応答を受け取る
- **出力の選択**: チャットに表示するワークフロー出力を選べる
- **ファイルアップロード**: 処理のためにメッセージへファイルを添付
- **リアルタイムストリーミング**: 生成される応答をリアルタイムに確認
- **メッセージ履歴**: 会話履歴を閲覧・スクロール

## 使い方

Workflow Chat ウィジェットは次のような用途に役立ちます：

- チャットベースのトリガーを持つワークフローのテスト
- ワークフロー内の AI エージェントとの対話
- ワークフローの応答をデバッグ
- ワークフローの機能をデモ

### メッセージの送信

1. 入力フィールドにメッセージを入力
2. 必要に応じてアップロードボタンでファイルを添付
3. Enter を押すか送信をクリックして送信
4. 生成中のレスポンスをリアルタイムで確認

### 出力の選択

チャットのレスポンスに表示するブロック出力を選択できます。ワークフローに複数の出力ブロックがあり、特定の結果に絞りたい場合に便利です。

<Callout>
このウィジェットは **pairColor** メカニズムをサポートしています。Workflow Editor と組み合わせると、編集中のワークフローとチャットできます。
</Callout>
