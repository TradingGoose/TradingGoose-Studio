ワークフロー変数ウィジェットは、ワークフローで定義された変数を表示および編集するための専用インターフェースを提供します。

## 概要

| プロパティ | 値 |
|----------|-------|
| **キー** | `workflow_variables` |
| **カテゴリ** | ユーティリティ |
| **アイコン** | 中括弧 |

## 機能

- **変数リスト**: すべてのワークフロー変数を一目で確認
- **インライン編集**: 変数値を直接編集
- **型表示**: 各変数のデータ型を確認
- **追加/削除**: 新しい変数の作成または既存の変数の削除
- **検索**: 名前で変数をフィルタリング

## 使用方法

ワークフロー変数ウィジェットは以下の用途に役立ちます：

- ワークフロー変数の現在の状態を検査する
- テスト用に変数値を変更する
- ワークフロー内のデータフローを理解する
- ワークフロースコープの設定を管理する

### 変数の型

ワークフロー変数は異なる種類のデータを保持できます：

| 型 | 説明 |
|------|-------------|
| **文字列** | テキスト値 |
| **数値** | 数値 |
| **ブール値** | 真/偽の値 |
| **オブジェクト** | JSONオブジェクト |
| **配列** | 値のリスト |

### 変数の編集

1. 変数をクリックして選択
2. エディタで値を変更
3. 変更は自動的に保存される
4. 次回実行時にワークフローは更新された値を使用

### 変数の作成

1. 「変数を追加」ボタンをクリック
2. 変数の名前を入力
3. 変数の型を選択
4. 初期値を設定（任意）

<Callout>
このウィジェットは**pairColor**メカニズムをサポートしています。ワークフローエディタとペアにすると、現在編集中のワークフローの変数を表示できます。
</Callout>
