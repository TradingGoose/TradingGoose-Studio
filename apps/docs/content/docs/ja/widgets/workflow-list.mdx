ワークフローリストウィジェットは、すべてのワークフローを階層的なフォルダツリーで表示し、ワークフローコレクションの整理、ナビゲーション、管理を可能にします。

## 概要

| プロパティ | 値 |
|----------|-------|
| **キー** | `workflow_list` |
| **カテゴリ** | リスト |
| **アイコン** | ListTree |

## 機能

- **フォルダツリー**: ワークフローの階層的な整理
- **ドラッグアンドドロップ**: フォルダ間でワークフローを移動
- **クイックナビゲーション**: クリックしてワークフローを開く
- **コンテキストメニュー**: 右クリックで追加アクションにアクセス
- **検索**: 名前でワークフローをフィルタリング
- **新規作成**: 新しいワークフローとフォルダの追加

## 使用方法

ワークフローリストウィジェットはサイドバーナビゲーションを反映し、ワークフローを管理するための専用ビューを提供します。

### 整理

- **フォルダ**: 関連するワークフローをグループ化するフォルダを作成
- **ドラッグアンドドロップ**: アイテムを新しい場所にドラッグして移動
- **名前変更**: ダブルクリックまたはコンテキストメニューを使用して名前を変更
- **削除**: ワークフローまたは空のフォルダを削除

### コンテキストメニューアクション

ワークフローまたはフォルダを右クリックして以下にアクセス：

| アクション | 説明 |
|--------|-------------|
| **開く** | エディタでワークフローを開く |
| **名前変更** | 名前を変更する |
| **複製** | コピーを作成する |
| **移動先** | 別のフォルダに移動する |
| **削除** | アイテムを削除する |

### ワークフローの選択

ワークフローをクリックして選択します。同じ色を使用する他のウィジェットとペアリングすると、選択したワークフローがそれらのウィジェットに表示されます。

<Callout>
このウィジェットは**pairColor**メカニズムをサポートしています。このリストでワークフローを選択すると、ペアリングされた他のウィジェットは自動的に切り替わり、そのワークフローを表示します。
</Callout>
