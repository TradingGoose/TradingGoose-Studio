ワークフローコパイロットウィジェットは、ワークフローの構築と最適化を支援するために特別に設計されたAI搭載アシスタント体験を提供します。

## 概要

| プロパティ | 値 |
|----------|-------|
| **キー** | `workflow_copilot` |
| **カテゴリ** | ユーティリティ |
| **アイコン** | スパークル |

## 機能

- **AI搭載アシスタンス**: ワークフロー設計のためのインテリジェントなサポート
- **コンテキスト認識**: 現在のワークフロー構造を理解
- **ブロック提案**: 目標に基づいたブロックの推奨
- **トラブルシューティング**: ワークフローの問題診断と修正の支援
- **ベストプラクティス**: ワークフローを最適化するための提案

## 使用方法

ワークフローコパイロットは以下の点でサポートします：

- 新しいワークフローをゼロから設計
- 既存のワークフローの最適化
- ブロック設定の理解
- 実行問題のデバッグ
- TradingGoose機能の学習

### 対話例

**新しいワークフローの作成：**
> 「GitHubリポジトリを監視し、新しい問題に対してSlack通知を送信するワークフローを作成して」

**ブロックの理解：**
> 「ルーターブロックは何をするもので、いつ使用すべきですか？」

**デバッグ：**
> 「なぜワークフローがAPIブロックで失敗するのですか？」

<Callout>
このウィジェットは**pairColor**メカニズムをサポートしています。ワークフローエディターとペアにすると、作業中の特定のワークフローに対するアシスタンスを受けられます。
</Callout>
