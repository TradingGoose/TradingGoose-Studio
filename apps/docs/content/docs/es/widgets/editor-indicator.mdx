## Descripción general

| Propiedad | Valor |
|----------|-------|
| **Clave** | `editor_indicator` |
| **Categoría** | Editor |

El Editor de Indicadores es donde escribes y perfeccionas indicadores PineTS personalizados. Proporciona un editor de código Monaco completo con soporte para TypeScript, una hoja de referencia rápida integrada para las variables globales de PineTS, y verificación del lado del servidor para que puedas validar tu script antes de guardarlo. Utiliza el widget [Lista de Indicadores](/indicators) para explorar y crear indicadores; el editor es el espacio de trabajo principal para escribir su lógica.

## Características

- **Editor de código Monaco** con resaltado de sintaxis TypeScript, autocompletado y las definiciones de tipos de PineTS cargadas automáticamente.
- **Barra de herramientas de hoja de referencia** agrupada por categoría (datos, análisis técnico, entrada, gráfico y más). Pasa el cursor sobre cualquier símbolo para ver su firma, descripción y ejemplos de uso.
- **Botón Verificar** ejecuta el indicador del lado del servidor con datos simulados e informa el número de gráficos y marcadores generados, además de advertencias sobre estilos o tipos de gráficos no compatibles.
- **Botón Guardar** persiste el código PineTS e infiere automáticamente `inputMeta` a partir de las llamadas `input.*` en el script, de modo que el gráfico pueda renderizar los controles de entrada.
- **Barra de solicitudes Wand** permite describir la lógica del indicador en lenguaje natural y transmite el código PineTS generado directamente al editor.
- **Menú desplegable de variables de entorno** se activa al escribir el patrón `{{`, permitiéndote referenciar secretos del espacio de trabajo dentro del código del indicador.
- **Selector de indicadores** en la barra de encabezado permite cambiar entre indicadores en el espacio de trabajo actual. Cuando se vincula mediante un enlace de color, el selector se sincroniza con el widget Lista de Indicadores.

## Uso

1. Agrega el widget a un espacio de trabajo desde el selector de widgets.
2. Selecciona un indicador existente desde el menú desplegable del encabezado, o crea uno primero en el widget [Lista de Indicadores](/indicators).
3. Escribe código PineTS en el editor. Usa las variables globales directamente -- `close`, `ta.sma`, `input.int`, `plot` -- sin el prefijo `$`.
4. Haz clic en el botón **Verificar** (ícono de marca de verificación) para ejecutar una prueba en seco del lado del servidor. Corrige cualquier error o advertencia que aparezca.
5. Haz clic en el botón **Guardar** para persistir el indicador. El gráfico tomará el código actualizado en el próximo renderizado.

### Atajos de Teclado

- **Activar/desactivar varita**: Haz clic en el ícono de varita en el margen del editor para abrir o cerrar la barra de prompts de IA.

<Callout>
Este widget admite el mecanismo **pairColor**. Asigna el mismo color a un Editor de Indicadores y a un widget [Lista de Indicadores](/indicators) para que al seleccionar un indicador en la lista se abra automáticamente en el editor.
</Callout>

## Configuración

El widget no tiene configuración visible para el usuario más allá del selector de indicadores. El indicador seleccionado se almacena en los parámetros del widget como `pineIndicatorId` y se sincroniza a través del almacén de color de par cuando está vinculado.
