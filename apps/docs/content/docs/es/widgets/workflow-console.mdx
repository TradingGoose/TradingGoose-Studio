El widget de Consola de flujo de trabajo proporciona una vista en tiempo real de la ejecución del flujo de trabajo, mostrando registros, información de depuración y salida en streaming mientras se ejecuta tu flujo de trabajo.

## Descripción general

| Propiedad | Valor |
|----------|-------|
| **Clave** | `workflow_console` |
| **Categoría** | Utilidad |
| **Icono** | Actividad |

## Características

- **Registros en tiempo real**: Visualiza los registros de ejecución mientras ocurren
- **Salida en streaming**: Observa las salidas de los bloques en tiempo real
- **Estado de ejecución**: Monitorea el estado actual de la ejecución del flujo de trabajo
- **Visualización de errores**: Identifica y depura errores rápidamente
- **Filtrado de registros**: Filtra registros por tipo, bloque o gravedad
- **Limpiar consola**: Reinicia la salida de la consola

## Uso

La Consola de flujo de trabajo es esencial para:

- Depurar la ejecución del flujo de trabajo
- Monitorear flujos de trabajo de larga duración
- Ver salidas intermedias de bloques
- Identificar errores y cuellos de botella

### Tipos de registros

| Tipo | Descripción |
|------|-------------|
| **Info** | Información general de ejecución |
| **Debug** | Información detallada de depuración |
| **Warning** | Problemas no críticos |
| **Error** | Errores y fallos de ejecución |
| **Output** | Datos de salida de bloques |

### Visualización del flujo de ejecución

La consola muestra la ruta de ejecución a través de tu flujo de trabajo:

1. **Inicio**: Comienza la ejecución del flujo de trabajo
2. **Entrada de bloque**: Cada bloque comienza el procesamiento
3. **Salida de bloque**: Resultados de cada bloque
4. **Salida de bloque**: El bloque se completa
5. **Fin**: La ejecución del flujo de trabajo se completa

<Callout>
Este widget admite el mecanismo **pairColor**. Emparéjalo con un Editor de flujo de trabajo para ver los registros del flujo de trabajo que estás editando.
</Callout>
