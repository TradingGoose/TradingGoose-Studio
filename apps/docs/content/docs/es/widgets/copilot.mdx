## Descripción general

| Propiedad | Valor |
|----------|-------|
| **Clave** | `copilot` |
| **Categoría** | Utilidad |

El widget Copilot es un asistente de IA de alcance global en el espacio de trabajo. Puede gestionar flujos de trabajo, servidores MCP, habilidades, herramientas personalizadas, bases de conocimiento, despliegues, variables y más. Mantiene su propio historial de chat por canal y admite múltiples conversaciones simultáneas.

## Funcionalidades

- **Asistente de IA de alcance global** — no limitado a flujos de trabajo. Copilot puede crear y editar servidores MCP, habilidades, herramientas personalizadas, consultar bases de conocimiento, desplegar flujos de trabajo, establecer variables y ejecutar flujos de trabajo
- **Consciente del contexto** — cuando se empareja con un panel de edición (flujo de trabajo, indicador, habilidad, MCP o herramienta personalizada), Copilot incluye automáticamente ese contexto para poder referenciar y modificar los recursos correctos
- **Historial de chat persistente** — las conversaciones se almacenan por canal y se agrupan por recencia (Hoy, Ayer, Esta semana, Anterior) en un menú desplegable de historial
- **Múltiples chats** — crea nuevas sesiones con **+** en el encabezado; cambia entre ellas sin perder conversaciones anteriores
- **Canales con alcance por panel** — cuando se empareja mediante color de emparejamiento, cada panel obtiene su propio canal para que múltiples widgets Copilot operen de forma independiente

## Uso

### Agregar el widget

Abre el selector de widgets y selecciona **Copilot**. Se inicializa una sesión de chat con alcance al contexto del espacio de trabajo actual.

### Disposición del encabezado

- **Izquierda** — menú desplegable del historial de chat; haz clic para explorar y cambiar entre conversaciones anteriores
- **Derecha** — botón de nuevo chat (**+**) para iniciar una conversación nueva

### Interacciones principales

1. Escribe una pregunta o instrucción en el área de entrada y envíala
2. Copilot transmite su respuesta en tiempo real, ejecutando llamadas a herramientas para gestionar los recursos del espacio de trabajo según sea necesario
3. Abre el menú desplegable de historial para revisar o eliminar conversaciones anteriores
4. Haz clic en **+** para iniciar un nuevo hilo de chat mientras mantienes los anteriores accesibles

### Sincronización de color de par

Cuando se le asigna un color de par, Copilot sigue el contexto del editor emparejado, ya sea un flujo de trabajo, indicador, habilidad, servidor MCP o herramienta personalizada. Cada panel mantiene su propio canal, por lo que múltiples widgets de Copilot con el mismo color de par operan de forma independiente.
