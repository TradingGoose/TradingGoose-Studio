## Descripción general

| Propiedad | Valor |
|----------|-------|
| **Clave** | `workflow_chat` |
| **Categoría** | Utilidad |

El widget Workflow Chat proporciona una interfaz conversacional para interactuar con los bloques del flujo de trabajo. Se conecta al flujo de trabajo activo a través de un socket en vivo, transmite respuestas en tiempo real y permite seleccionar qué salidas de bloque aparecen en la conversación. Úselo para probar disparadores basados en chat, comunicarse con agentes de IA dentro de un flujo de trabajo o inspeccionar respuestas durante el desarrollo.

## Funcionalidades

- **Conexión por socket en vivo** -- cada instancia de chat abre su propio socket al flujo de trabajo, de modo que los mensajes y respuestas se transmiten en tiempo real
- **Selector de salidas** -- un menú desplegable de selección múltiple en el encabezado permite elegir qué salidas de bloque del flujo de trabajo se incluyen en la respuesta del chat; las selecciones se deduplicан y se persisten por flujo de trabajo
- **Limpiar chat** -- un botón en el encabezado borra el historial de conversación actual del flujo de trabajo activo; el botón se deshabilita cuando no hay mensajes
- **Selector de flujo de trabajo** -- un menú desplegable en el centro del encabezado permite cambiar entre flujos de trabajo del espacio de trabajo; cuando el widget está emparejado (color de par no gris), el menú refleja la selección compartida y los cambios manuales se ignoran
- **Resolución automática del flujo de trabajo** -- el widget resuelve el flujo de trabajo activo a partir de los parámetros del widget, el canal de color de par o el registro de flujos de trabajo, y persiste la selección entre sesiones

## Uso

### Agregar el widget

Abra el selector de widgets y seleccione **Workflow Chat**. El widget requiere un espacio de trabajo activo; si no hay ninguno seleccionado, muestra un mensaje para elegir uno. Una vez cargado el espacio de trabajo, se conecta automáticamente al primer flujo de trabajo disponible.

### Diseño del encabezado

- **Izquierda** -- menú desplegable de selector de salida para elegir qué salidas de bloque se incorporan al chat
- **Centro** -- menú desplegable de flujo de trabajo para cambiar el flujo de trabajo de destino
- **Derecha** -- botón para limpiar el chat (icono de papelera) para reiniciar la conversación

### Interacciones clave

1. Escribe un mensaje en el campo de entrada en la parte inferior y presiona Enter para enviar
2. Las respuestas se transmiten al área de conversación a medida que se ejecuta el flujo de trabajo
3. Usa el selector de salida para filtrar las respuestas a bloques específicos
4. Haz clic en el botón de limpiar para iniciar una conversación nueva

### Sincronización de color de par

Cuando se le asigna un color de par, el widget comparte su flujo de trabajo activo con otros widgets en el mismo canal de color -- típicamente el Editor de flujo de trabajo. Cambiar el flujo de trabajo en el editor actualiza automáticamente este widget de chat. Cuando el color de par es `gray`, el widget gestiona su propia selección de flujo de trabajo de forma independiente.

<Callout>
Empareja este widget con el **Editor de flujo de trabajo** usando un color de par coincidente para que el chat siempre apunte al flujo de trabajo que estás editando.
</Callout>
