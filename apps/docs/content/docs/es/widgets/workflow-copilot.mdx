El widget Copiloto de flujo de trabajo proporciona una experiencia de asistente con tecnología de IA específicamente diseñada para ayudarte a crear y optimizar tus flujos de trabajo.

## Descripción general

| Propiedad | Valor |
|----------|-------|
| **Clave** | `workflow_copilot` |
| **Categoría** | Utilidad |
| **Icono** | Destellos |

## Características

- **Asistencia con tecnología de IA**: Obtén ayuda inteligente para el diseño de flujos de trabajo
- **Sensible al contexto**: Comprende la estructura actual de tu flujo de trabajo
- **Sugerencias de bloques**: Recomienda bloques basados en tus objetivos
- **Solución de problemas**: Ayuda a diagnosticar y solucionar problemas de flujo de trabajo
- **Mejores prácticas**: Sugerencias para optimizar tus flujos de trabajo

## Uso

El Copiloto de flujo de trabajo puede ayudarte con:

- Diseñar nuevos flujos de trabajo desde cero
- Optimizar flujos de trabajo existentes
- Comprender configuraciones de bloques
- Depurar problemas de ejecución
- Aprender características de TradingGoose

### Ejemplos de interacciones

**Crear un nuevo flujo de trabajo:**
> "Crea un flujo de trabajo que monitoree un repositorio de GitHub y envíe notificaciones de Slack para nuevos problemas"

**Comprender bloques:**
> "¿Qué hace el bloque Router y cuándo debería usarlo?"

**Depuración:**
> "¿Por qué mi flujo de trabajo está fallando en el bloque API?"

<Callout>
Este widget admite el mecanismo **pairColor**. Emparéjalo con un Editor de flujo de trabajo para obtener asistencia para el flujo de trabajo específico en el que estás trabajando.
</Callout>
