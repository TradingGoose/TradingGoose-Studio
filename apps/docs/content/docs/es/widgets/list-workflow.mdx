## Descripción General

| Propiedad | Valor |
|----------|-------|
| **Clave** | `list_workflow` |
| **Categoría** | Lista |

El widget Lista de Flujos de Trabajo muestra una lista navegable de todos los flujos de trabajo en tu espacio de trabajo.

## Características

- **Explorar**: Ver todos los flujos de trabajo en tu espacio de trabajo
- **Buscar y Filtrar**: Encuentra flujos de trabajo rápidamente
- **Crear Nuevo**: Inicia un nuevo flujo de trabajo directamente desde la lista
- **Abrir en el Editor**: Haz clic para abrir cualquier flujo de trabajo en el widget editor

## Uso

Añade este widget junto al Editor de Flujos de Trabajo para gestionar tus flujos de trabajo.
