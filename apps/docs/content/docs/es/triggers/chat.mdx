El disparador de Chat crea una interfaz conversacional para tu flujo de trabajo. Despliega tu flujo de trabajo como un chat y los usuarios podrán interactuar con él a través de una URL compartible. Cada mensaje inicia una nueva ejecución del flujo de trabajo.

<Callout type="info">
  Este es un disparador **basado en despliegue**. Los usuarios interactúan con tu flujo de trabajo a través de una URL de chat alojada.
</Callout>

### Esquema de Salida

| Campo | Tipo | Descripción |
|-------|------|-------------|
| `input` | string | Último mensaje del usuario |
| `conversationId` | string | ID del hilo de conversación |
| `files` | files | Archivos adjuntos opcionales |

### Propiedades del archivo

| Propiedad | Tipo | Descripción |
|----------|------|-------------|
| `<chat.files[0].url>` | string | URL de descarga firmada |
| `<chat.files[0].name>` | string | Nombre de archivo original |
| `<chat.files[0].size>` | number | Tamaño del archivo en bytes |
| `<chat.files[0].type>` | string | Tipo MIME |

<Callout>
Cada conversación mantiene su propio hilo. Los usuarios pueden tener múltiples conversaciones simultáneas.
</Callout>
