El activador Manual añade un botón Ejecutar a tu flujo de trabajo. Úsalo cuando desees ejecutar el flujo de trabajo de inmediato sin recopilar datos adicionales.

<Callout type="info">
  Este es un activador **manual**. No se requiere configuración — simplemente haz clic en Ejecutar.
</Callout>

## Configuración

No se requiere configuración. El activador manual no tiene campos de entrada ni produce salidas.

## Cuándo Usarlo

- Verificaciones rápidas de integridad y pruebas de humo
- Flujos de trabajo que dependen únicamente de variables internas o llamadas a API externas
- Desarrollo y depuración

<Callout>
¿Necesitas una entrada estructurada? Utiliza un activador de Formulario de Entrada en su lugar.
</Callout>
