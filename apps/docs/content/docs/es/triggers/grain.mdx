Activa el flujo de trabajo cuando se actualiza una grabación en Grain

<Callout type="info">
  Este es un disparador **basado en webhook**. Configure la URL del webhook en su servicio externo para enviar eventos a TradingGoose.
</Callout>

## Eventos

### Grabación de Grain Actualizada

Activa el flujo de trabajo cuando se actualiza una grabación en Grain

#### Esquema de Salida

El disparador pasa el payload completo del evento a su flujo de trabajo. Acceda a los campos utilizando la sintaxis `<trigger.fieldName>`.
