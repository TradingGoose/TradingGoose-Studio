Las conexiones son los canales que permiten que los datos fluyan entre bloques en tu flujo de trabajo. Definen cómo se transfiere la información de un bloque a otro, lo que te permite crear procesos sofisticados de múltiples pasos.

<Callout type="info">
  Las conexiones correctamente configuradas son esenciales para crear flujos de trabajo eficaces. Determinan cómo
  se mueven los datos a través de tu sistema y cómo interactúan los bloques entre sí.
</Callout>

Para crear una conexión, arrastra desde el puerto de salida de un bloque hasta el puerto de entrada de otro. Una vez conectados, los datos producidos por el bloque de origen quedan disponibles en el bloque de destino mediante etiquetas de conexión.
