## Herramientas

### `trading_order_detail`

Recupera todos los detalles disponibles del proveedor para una orden de trading previamente registrada, mediante el ID de orden de Trading Goose.

#### Entrada

| Parámetro | Tipo | Requerido | Descripción |
| --------- | ---- | -------- | ----------- |
| `orderId` | string | Sí | ID de orden de Trading Goose \(orderHistoryTable.id\) generado al enviar la orden. |
| `provider` | string | No | Proveedor esperado para esta orden. Se utiliza para la selección de credenciales y la validación de discrepancias. |
| `environment` | string | No | Anulación opcional del entorno \(paper o live\) para la obtención de detalles de la orden del proveedor. |
| `credential` | string | No | ID de credencial OAuth para el bróker seleccionado \(completado desde la cuenta seleccionada\). |
| `tradierCredential` | string | No | ID de credencial OAuth de Tradier. |
| `robinhoodCredential` | string | No | ID de credencial OAuth de Robinhood. |
| `alpacaCredential` | string | No | ID de credencial OAuth de Alpaca. |
| `apiKey` | string | No | ID de clave API de Alpaca \(opcional si se utiliza OAuth\). |
| `apiSecret` | string | No | Clave secreta de API de Alpaca \(opcional si se utiliza OAuth\). |
| `accountId` | string | No | Anulación del ID de cuenta de Tradier si no está presente en los metadatos de la orden almacenada. |
| `accountUrl` | string | No | Anulación de la URL de cuenta de Robinhood si es necesario. |

#### Salida

| Parámetro | Tipo | Descripción |
| --------- | ---- | ----------- |
| `summary` | string | Mensaje de estado para la recuperación de detalles de la orden. |
| `provider` | string | Bróker/proveedor de la orden. |
| `appOrderId` | string | ID de orden de Trading Goose \(orderHistoryTable.id\). |
| `providerOrderId` | string | ID de orden nativo del proveedor utilizado para la consulta a la API. |

## Instrucciones de uso

Busca el registro del historial de órdenes de Trading Goose por ID de orden, resuelve el ID de orden del proveedor y obtiene el detalle más reciente de la orden del proveedor.
