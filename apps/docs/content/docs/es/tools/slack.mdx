<div className="text-base font-semibold mb-2">Enviar mensaje (<code>slack_message</code>)</div>

Envía mensajes a canales de Slack o mensajes directos. Compatible con el formato mrkdwn de Slack.

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">Entrada</div>

| Parámetro | Tipo | Requerido | Descripción |
| --------- | ---- | -------- | ----------- |
| `authMethod` | string | No | Método de autenticación: oauth o bot_token |
| `destinationType` | string | No | Tipo de destino: channel o dm |
| `botToken` | string | No | Token de bot para bot personalizado |
| `channel` | string | No | Canal de Slack de destino \(p. ej., #general\) |
| `dmUserId` | string | No | Usuario de Slack de destino para mensajes directos |
| `text` | string | Sí | Texto del mensaje a enviar \(compatible con formato mrkdwn de Slack\) |
| `thread_ts` | string | No | Marca de tiempo del hilo al que responder \(crea una respuesta en hilo\) |
| `files` | file[] | No | Archivos para adjuntar al mensaje |

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">Salida</div>

| Parámetro | Tipo | Descripción |
| --------- | ---- | ----------- |
| `message` | object | Objeto de mensaje completo con todas las propiedades devueltas por Slack |
| `ts` | string | Marca de tiempo del mensaje |
| `channel` | string | ID del canal donde se envió el mensaje |

</div>

## Instrucciones de uso

Integra Slack en el flujo de trabajo. Permite enviar mensajes, crear lienzos y leer mensajes. Requiere Token de Bot en lugar de OAuth en modo avanzado. Puede utilizarse en modo disparador para iniciar un flujo de trabajo cuando se envía un mensaje a un canal.
