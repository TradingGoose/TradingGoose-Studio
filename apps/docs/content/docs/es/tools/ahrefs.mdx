<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">Entrada</div>

| Parámetro | Tipo | Requerido | Descripción |
| --------- | ---- | -------- | ----------- |
| `target` | string | Sí | El dominio o URL de destino a analizar |
| `mode` | string | No | Modo de análisis: domain \(dominio completo\), prefix \(prefijo de URL\), subdomains \(incluir todos los subdominios\), exact \(coincidencia exacta de URL\) |
| `date` | string | No | Fecha para datos históricos en formato YYYY-MM-DD \(por defecto: hoy\) |
| `limit` | number | No | Número máximo de resultados a devolver \(por defecto: 100\) |
| `offset` | number | No | Número de resultados a omitir para la paginación |
| `apiKey` | string | Sí | Clave API de Ahrefs |

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">Salida</div>

| Parámetro | Tipo | Descripción |
| --------- | ---- | ----------- |
| `urlFrom` | string | La URL de la página que contiene el backlink |
| `urlTo` | string | La URL a la que se enlaza |
| `anchor` | string | El texto ancla del enlace |
| `domainRatingSource` | number | Calificación de dominio del dominio enlazante |
| `isDofollow` | boolean | Si el enlace es dofollow |
| `firstSeen` | string | Cuándo fue descubierto el backlink por primera vez |
