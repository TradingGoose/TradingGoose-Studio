## Instrucciones de uso

Elija un proveedor de datos de mercado, seleccione un listado canónico y obtenga barras de series que incluyan apertura, máximo, mínimo, cierre, volumen y marcas de tiempo. El inicio y el fin pueden ser cadenas ISO o marcas de tiempo UNIX.
