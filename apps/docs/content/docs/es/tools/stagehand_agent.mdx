## Herramientas

### `stagehand_agent`

Ejecutar un agente web autónomo para completar tareas y extraer datos estructurados

#### Entrada

| Parámetro | Tipo | Requerido | Descripción |
| --------- | ---- | -------- | ----------- |
| `startUrl` | string | Sí | URL de la página web donde iniciar el agente |
| `task` | string | Sí | La tarea a completar o el objetivo a lograr en el sitio web |
| `variables` | json | No | Variables opcionales para sustituir en la tarea \(formato: \{key: value\}\). Referenciar en la tarea usando %key% |
| `format` | string | No | Sin descripción |
| `provider` | string | No | Proveedor de IA a utilizar: openai o anthropic |
| `apiKey` | string | Sí | Clave de API para el proveedor seleccionado |
| `outputSchema` | json | No | Esquema JSON opcional que define la estructura de datos que el agente debe retornar |

#### Salida

| Parámetro | Tipo | Descripción |
| --------- | ---- | ----------- |
| `success` | boolean | Si la tarea del agente se completó exitosamente |
| `completed` | boolean | Si la tarea fue completada en su totalidad |
| `message` | string | Mensaje de estado o resultado final |

## Instrucciones de uso

Integrar el Agente Stagehand en el flujo de trabajo. Puede navegar por la web y realizar tareas.
