## Herramientas

### `perplexity_chat`

Generar completaciones utilizando los modelos de chat de Perplexity AI

#### Entrada

| Parámetro | Tipo | Obligatorio | Descripción |
| --------- | ---- | -------- | ----------- |
| `systemPrompt` | string | No | Prompt del sistema para guiar el comportamiento del modelo |
| `content` | string | Sí | El contenido del mensaje del usuario a enviar al modelo |
| `model` | string | Sí | Modelo a utilizar para las completaciones de chat \(p. ej., sonar, mistral\) |
| `max_tokens` | number | No | Número máximo de tokens a generar |
| `temperature` | number | No | Temperatura de muestreo entre 0 y 1 |
| `apiKey` | string | Sí | Clave API de Perplexity |

#### Salida

| Parámetro | Tipo | Descripción |
| --------- | ---- | ----------- |
| `success` | boolean | Estado de éxito de la operación |
| `content` | string | Contenido de texto generado |
| `model` | string | Modelo utilizado para la generación |
| `prompt_tokens` | number | Número de tokens en el prompt |
| `completion_tokens` | number | Número de tokens en la completación |

## Instrucciones de uso

Integra Perplexity en el flujo de trabajo. Puede generar completaciones utilizando los modelos de chat de Perplexity AI.
