## Instrucciones de uso

Integra Webflow CMS en el flujo de trabajo. Permite crear, obtener, listar, actualizar o eliminar elementos en colecciones de Webflow CMS. Gestiona tu contenido de Webflow de forma programática. Puede utilizarse en modo de activación para iniciar flujos de trabajo cuando los elementos de una colección cambian o se envían formularios.
