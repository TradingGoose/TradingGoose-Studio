## Instrucciones de uso

Integre Amazon RDS Aurora Serverless en el flujo de trabajo mediante la API de datos. Permite consultar, insertar, actualizar, eliminar y ejecutar SQL sin procesar sin necesidad de gestionar conexiones de base de datos.
