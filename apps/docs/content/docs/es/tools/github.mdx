## Instrucciones de uso

Integra GitHub en el flujo de trabajo. Permite obtener detalles de PRs, crear comentarios en PRs, obtener información del repositorio y obtener el último commit. Puede utilizarse en modo trigger para activar un flujo de trabajo cuando se crea un PR, se comenta en uno o se realiza un push de un commit.
