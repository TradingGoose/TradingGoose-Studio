## Herramientas

### `trading_place_order`

#### Entrada

| Parámetro | Tipo | Requerido | Descripción |
| --------- | ---- | -------- | ----------- |

#### Salida

| Parámetro | Tipo | Descripción |
| --------- | ---- | ----------- |
| `summary` | string | Estado de envío de la orden |
| `provider` | string | Proveedor utilizado |
| `order` | json | Payload de la orden y respuesta sin procesar |

## Instrucciones de uso

Bloque de acción de trading unificado que admite múltiples brokerages con autenticación OAuth o mediante clave API.
