## Herramientas

### `resend_send`

Envía un correo electrónico utilizando tu propia clave API de Resend y dirección de origen

#### Entrada

| Parámetro | Tipo | Requerido | Descripción |
| --------- | ---- | -------- | ----------- |
| `fromAddress` | string | Sí | Dirección de correo electrónico desde la que se envía |
| `to` | string | Sí | Dirección de correo electrónico del destinatario |
| `subject` | string | Sí | Asunto del correo electrónico |
| `body` | string | Sí | Contenido del cuerpo del correo electrónico |
| `contentType` | string | No | Tipo de contenido del cuerpo del correo \(text o html\) |
| `resendApiKey` | string | Sí | Clave API de Resend para el envío de correos electrónicos |

#### Salida

| Parámetro | Tipo | Descripción |
| --------- | ---- | ----------- |
| `success` | boolean | Si el correo electrónico fue enviado correctamente |
| `to` | string | Dirección de correo electrónico del destinatario |
| `subject` | string | Asunto del correo electrónico |

## Instrucciones de uso

Integre Resend en el flujo de trabajo. Permite enviar correos electrónicos. Requiere clave de API.
