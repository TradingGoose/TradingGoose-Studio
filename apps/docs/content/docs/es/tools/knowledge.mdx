| `knowledgeBaseId` | string | Sí | ID de la base de conocimiento en la que buscar |
| `query` | string | No | Texto de la consulta de búsqueda \(opcional al usar filtros de etiquetas\) |
| `topK` | number | No | Número de resultados más similares a devolver \(1-100\) |
| `tagFilters` | array | No | Array de filtros de etiquetas con propiedades tagName y tagValue |
| `items` | object | No | Sin descripción |
| `properties` | string | No | Sin descripción |
| `tagName` | string | No | Sin descripción |
| `tagValue` | string | No | Sin descripción |
| `tagFilters` | string | No | Sin descripción |

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">Salida</div>

| Parámetro | Tipo | Descripción |
| --------- | ---- | ----------- |
| `documentId` | string | ID del documento |
| `documentName` | string | Nombre del documento |
| `content` | string | Contenido del resultado |
| `chunkIndex` | number | Índice del fragmento dentro del documento |
| `similarity` | number | Puntuación de similitud del resultado |
