## Instrucciones de uso

Genera vídeos de alta calidad a partir de indicaciones de texto utilizando los principales proveedores de IA. Compatible con múltiples modelos, relaciones de aspecto, resoluciones y funciones específicas de cada proveedor, como consistencia del mundo, controles de cámara y generación de audio.
