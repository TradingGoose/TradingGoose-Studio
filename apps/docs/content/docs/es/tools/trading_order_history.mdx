## Herramientas

### `trading_order_history`

Recupera las órdenes de envío registradas para un flujo de trabajo dentro de un rango de fechas y horas.

#### Entrada

| Parámetro | Tipo | Requerido | Descripción |
| --------- | ---- | -------- | ----------- |
| `startDate` | string | Sí | Fecha y hora de inicio \(ISO 8601\). |
| `endDate` | string | Sí | Fecha y hora de fin \(ISO 8601\). |
| `workflowId` | string | No | ID de flujo de trabajo opcional para filtrar. Por defecto, utiliza el contexto de ejecución del flujo de trabajo actual. |

#### Salida

| Parámetro | Tipo | Descripción |
| --------- | ---- | ----------- |
| `id` | string | ID del registro de historial de órdenes |
| `provider` | string | Proveedor de trading |
| `recordedAt` | string | Marca de tiempo registrada |
| `workflowId` | string | ID del flujo de trabajo |
| `workflowExecutionId` | string | ID de ejecución del flujo de trabajo |

## Instrucciones de Uso

Obtiene el historial de envío de órdenes registrado por la herramienta Trading Action para el flujo de trabajo y el rango de fechas seleccionados.
