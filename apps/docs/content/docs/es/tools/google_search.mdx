## Herramientas

### `google_search`

Busca en la web con la API de búsqueda personalizada

#### Entrada

| Parámetro | Tipo | Requerido | Descripción |
| --------- | ---- | -------- | ----------- |
| `query` | string | Sí | La consulta de búsqueda a ejecutar |
| `searchEngineId` | string | Sí | ID del motor de búsqueda personalizado |
| `num` | string | No | Número de resultados a devolver \(por defecto: 10, máx: 10\) |
| `apiKey` | string | Sí | Clave de API de Google |

#### Salida

| Parámetro | Tipo | Descripción |
| --------- | ---- | ----------- |
| `title` | string | Título del resultado de búsqueda |
| `link` | string | URL del resultado de búsqueda |
| `snippet` | string | Fragmento o descripción del resultado de búsqueda |
| `displayLink` | string | URL de visualización |

## Instrucciones de uso

Integra Google Search en el flujo de trabajo. Permite realizar búsquedas en la web.
