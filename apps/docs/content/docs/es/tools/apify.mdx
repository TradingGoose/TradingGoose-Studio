Ejecutar un actor de APIFY de forma síncrona y obtener resultados (máximo 5 minutos)

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">Entrada</div>

| Parámetro | Tipo | Requerido | Descripción |
| --------- | ---- | -------- | ----------- |
| `apiKey` | string | Sí | Token de API de APIFY desde console.apify.com/account#/integrations |
| `actorId` | string | Sí | ID del actor o nombre-de-usuario/nombre-del-actor \(p. ej., "janedoe/my-actor" o ID del actor\) |
| `input` | string | No | Entrada del actor como cadena JSON. Consulte la documentación del actor para los campos requeridos. |
| `timeout` | number | No | Tiempo de espera en segundos \(predeterminado: valor por defecto del actor\) |
| `build` | string | No | Compilación del actor a ejecutar \(p. ej., "latest", "beta" o etiqueta/número de compilación\) |

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">Salida</div>

| Parámetro | Tipo | Descripción |
| --------- | ---- | ----------- |
| `success` | boolean | Indica si la ejecución del actor fue exitosa |
| `runId` | string | ID de ejecución de APIFY |
| `status` | string | Estado de la ejecución \(SUCCEEDED, FAILED, etc.\) |

</div>

## Instrucciones de Uso

Integra Apify en tu flujo de trabajo. Ejecuta cualquier actor de Apify con entrada personalizada y recupera los resultados. Compatible con ejecución síncrona y asíncrona con obtención automática del dataset.
