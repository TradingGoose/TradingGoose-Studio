## Herramientas

### `trading_get_holdings`

#### Entrada

| Parámetro | Tipo | Requerido | Descripción |
| --------- | ---- | -------- | ----------- |

#### Salida

| Parámetro | Tipo | Descripción |
| --------- | ---- | ----------- |
| `summary` | string | Estado de la recuperación de posiciones |
| `provider` | string | Proveedor utilizado |
| `holdings` | json | Carga útil del snapshot unificado de la cuenta |

## Instrucciones de uso

Bloque de posiciones unificadas que devuelve un snapshot de cuenta para Alpaca, Tradier o Robinhood.
