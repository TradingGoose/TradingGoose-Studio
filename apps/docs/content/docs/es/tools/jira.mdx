Actualizar una incidencia de Jira

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">Entrada</div>

| Parámetro | Tipo | Requerido | Descripción |
| --------- | ---- | --------- | ----------- |
| `domain` | string | Sí | Tu dominio de Jira \(p. ej., tuempresa.atlassian.net\) |
| `projectId` | string | No | ID del proyecto de Jira en el que se actualizarán las incidencias. Si no se proporciona, se recuperarán todas las incidencias. |
| `issueKey` | string | Sí | Clave de la incidencia de Jira a actualizar |
| `summary` | string | No | Nuevo resumen para la incidencia |
| `description` | string | No | Nueva descripción para la incidencia |
| `status` | string | No | Nuevo estado para la incidencia |
| `priority` | string | No | Nueva prioridad para la incidencia |
| `assignee` | string | No | Nuevo asignado para la incidencia |
| `cloudId` | string | No | ID de nube de Jira para la instancia. Si no se proporciona, se obtendrá utilizando el dominio. |

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">Salida</div>

| Parámetro | Tipo | Descripción |
| --------- | ---- | ----------- |
| `ts` | string | Marca de tiempo de la operación |
| `issueKey` | string | Clave del issue actualizado \(p. ej., PROJ-123\) |

</div>

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">Entrada</div>

| Parámetro | Tipo | Requerido | Descripción |
| --------- | ---- | -------- | ----------- |
| `domain` | string | Sí | Tu dominio de Jira \(p. ej., tuempresa.atlassian.net\) |
| `projectId` | string | Sí | ID del proyecto para el issue |
| `summary` | string | Sí | Resumen del issue |
| `description` | string | No | Descripción del issue |
| `priority` | string | No | ID o nombre de prioridad del issue \(p. ej., "10000" o "High"\) |
| `assignee` | string | No | ID de cuenta del asignado del issue |
| `cloudId` | string | No | ID de Jira Cloud para la instancia. Si no se proporciona, se obtendrá usando el dominio. |
| `issueType` | string | Sí | Tipo de issue a crear \(p. ej., Task, Story\) |
| `labels` | array | No | Etiquetas del issue \(array de nombres de etiquetas\) |
| `duedate` | string | No | Fecha de vencimiento del issue \(formato: YYYY-MM-DD\) |
| `reporter` | string | No | ID de cuenta del reportero del issue |
| `environment` | string | No | Información de entorno del issue |
| `customFieldId` | string | No | ID de campo personalizado \(p. ej., customfield_10001\) |
| `customFieldValue` | string | No | Valor para el campo personalizado |

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">Salida</div>

| Parámetro | Tipo | Descripción |
| --------- | ---- | ----------- |
| `ts` | string | Marca de tiempo de la operación |
| `issueKey` | string | Clave del issue creado \(p. ej., PROJ-123\) |
| `summary` | string | Resumen del issue |
| `url` | string | URL del issue creado |

</div>
