<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">Entrada</div>

| Parámetro | Tipo | Requerido | Descripción |
| --------- | ---- | -------- | ----------- |
| `apiKey` | string | Sí | Clave de API de Cursor |
| `repository` | string | Sí | URL del repositorio de GitHub \(p. ej., https://github.com/your-org/your-repo\) |
| `ref` | string | No | Rama, etiqueta o commit desde el que trabajar \(por defecto, la rama predeterminada\) |
| `promptText` | string | Sí | El texto de instrucción para el agente |
| `promptImages` | string | No | Array JSON de objetos de imagen con datos en base64 y dimensiones |
| `model` | string | No | Modelo a utilizar \(dejar vacío para selección automática\) |
| `branchName` | string | No | Nombre de rama personalizado para que utilice el agente |
| `autoCreatePr` | boolean | No | Crear automáticamente un PR cuando el agente finalice |
| `openAsCursorGithubApp` | boolean | No | Abrir el PR como la aplicación GitHub de Cursor |
| `skipReviewerRequest` | boolean | No | Omitir la solicitud de revisores en el PR |

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">Salida</div>

| Parámetro | Tipo | Descripción |
| --------- | ---- | ----------- |
| `content` | string | Mensaje de éxito con detalles del agente |
| `id` | string | ID del agente |
