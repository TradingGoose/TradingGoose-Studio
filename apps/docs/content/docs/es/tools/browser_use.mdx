## Herramientas

### `browser_use_run_task`

Ejecuta una tarea de automatización del navegador utilizando BrowserUse

#### Entrada

| Parámetro | Tipo | Requerido | Descripción |
| --------- | ---- | -------- | ----------- |
| `task` | string | Sí | Qué debe hacer el agente del navegador |
| `variables` | json | No | Variables opcionales para usar como secretos \(formato: \{key: value\}\) |
| `format` | string | No | Sin descripción |
| `save_browser_data` | boolean | No | Si se deben guardar los datos del navegador |
| `model` | string | No | Modelo LLM a utilizar \(predeterminado: gpt-4o\) |
| `apiKey` | string | Sí | Clave API para la API de BrowserUse |
| `profile_id` | string | No | ID de perfil del navegador para sesiones persistentes \(cookies, estado de sesión\) |

#### Salida

| Parámetro | Tipo | Descripción |
| --------- | ---- | ----------- |
| `id` | string | Identificador de ejecución de la tarea |
| `success` | boolean | Estado de finalización de la tarea |
| `output` | json | Datos de salida de la tarea |

## Instrucciones de Uso

Integra Browser Use en el flujo de trabajo. Puede navegar por la web y realizar acciones como si un usuario real estuviera interactuando con el navegador.
