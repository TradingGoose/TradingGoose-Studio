## Herramientas

### `mistral_parser`

Analizar documentos PDF mediante la API OCR de Mistral

#### Entrada

| Parámetro | Tipo | Requerido | Descripción |
| --------- | ---- | -------- | ----------- |
| `filePath` | string | Sí | URL de un documento PDF a procesar |
| `fileUpload` | object | No | Datos de carga de archivo desde el componente de subida de archivos |
| `resultType` | string | No | Tipo de resultado analizado \(markdown, text o json\). Por defecto: markdown. |
| `includeImageBase64` | boolean | No | Incluir imágenes codificadas en base64 en la respuesta |
| `pages` | array | No | Páginas específicas a procesar \(array de números de página, comenzando desde 0\) |
| `imageLimit` | number | No | Número máximo de imágenes a extraer del PDF |
| `imageMinSize` | number | No | Altura y anchura mínimas de las imágenes a extraer del PDF |
| `apiKey` | string | Sí | Clave de API de Mistral \(MISTRAL_API_KEY\) |

#### Salida

| Parámetro | Tipo | Descripción |
| --------- | ---- | ----------- |
| `success` | boolean | Indica si el PDF fue analizado correctamente |
| `content` | string | Contenido extraído en el formato solicitado \(markdown, text o JSON\) |
| `jobId` | string | Identificador único del trabajo |
| `fileType` | string | Tipo de archivo \(p. ej., pdf\) |
| `fileName` | string | Nombre original del archivo |
| `source` | string | Tipo de fuente \(url\) |
| `pageCount` | number | Número de páginas procesadas |
| `model` | string | Modelo de Mistral utilizado |
| `resultType` | string | Formato de salida \(markdown, text, json\) |
| `processedAt` | string | Marca de tiempo del procesamiento |
| `sourceUrl` | string | URL de origen si aplica |

## Instrucciones de uso

Integre Mistral Parse en el flujo de trabajo. Puede extraer texto de documentos PDF cargados o desde una URL.
