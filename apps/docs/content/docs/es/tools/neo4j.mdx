| `parameters` | object | No | Parámetros para la consulta Cypher como objeto JSON. Utilícelo para cualquier valor dinámico, incluido LIMIT \(p. ej., query: "MATCH \(n\) RETURN n LIMIT $limit", parameters: \{limit: 100\}\). |
| `parameters` | string | No | Sin descripción |
