## Herramientas

### `google_forms_get_responses`

#### Entrada

| Parámetro | Tipo | Requerido | Descripción |
| --------- | ---- | -------- | ----------- |

#### Salida

| Parámetro | Tipo | Descripción |
| --------- | ---- | ----------- |
| `data` | json | Respuesta o lista de respuestas |

## Instrucciones de uso

Integra Google Forms en tu flujo de trabajo. Proporciona un ID de formulario para listar respuestas, o especifica un ID de respuesta para obtener una respuesta individual. Requiere OAuth.
