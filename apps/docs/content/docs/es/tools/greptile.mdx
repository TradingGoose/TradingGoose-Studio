<div className="text-base font-semibold mb-2">Consulta (<code>greptile_query</code>)</div>

Consulte repositorios en lenguaje natural y obtenga respuestas con referencias de código relevantes. Greptile utiliza IA para comprender su código base y responder preguntas.

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">Entrada</div>

| Parámetro | Tipo | Requerido | Descripción |
| --------- | ---- | -------- | ----------- |
| `query` | string | Sí | Pregunta en lenguaje natural sobre el código base |
| `repositories` | string | Sí | Lista de repositorios separados por comas. Formato: "github:branch:owner/repo" o simplemente "owner/repo" \(por defecto: github:main\) |
| `sessionId` | string | No | ID de sesión para continuidad de la conversación |
| `genius` | boolean | No | Activar el modo Genius para un análisis más exhaustivo \(más lento pero más preciso\) |
| `apiKey` | string | Sí | Clave API de Greptile |
| `githubToken` | string | Sí | Token de acceso personal de GitHub con permisos de lectura del repositorio |

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">Salida</div>

| Parámetro | Tipo | Descripción |
| --------- | ---- | ----------- |
| `message` | string | Respuesta generada por IA a la consulta |
| `repository` | string | Nombre del repositorio \(owner/repo\) |
| `remote` | string | Remoto de Git \(github/gitlab\) |
| `branch` | string | Nombre de la rama |
| `filepath` | string | Ruta al archivo |
| `linestart` | number | Número de línea inicial |
| `lineend` | number | Número de línea final |
| `summary` | string | Resumen de la sección de código |

</div>

<div className="text-base font-semibold mb-2">Indexar Repositorio (<code>greptile_index_repo</code>)</div>

Envía un repositorio para ser indexado por Greptile. La indexación debe completarse antes de que el repositorio pueda ser consultado. Los repositorios pequeños tardan entre 3 y 5 minutos; los más grandes pueden tardar más de una hora.

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">Entrada</div>

| Parámetro | Tipo | Requerido | Descripción |
| --------- | ---- | -------- | ----------- |
| `remote` | string | Sí | Tipo de remoto Git: github o gitlab |
| `repository` | string | Sí | Repositorio en formato propietario/repo \(p. ej., "facebook/react"\) |
| `branch` | string | Sí | Rama a indexar \(p. ej., "main" o "master"\) |
| `reload` | boolean | No | Forzar la re-indexación aunque ya esté indexado |
| `notify` | boolean | No | Enviar notificación por correo electrónico al completar la indexación |
| `apiKey` | string | Sí | Clave de API de Greptile |
| `githubToken` | string | Sí | Token de Acceso Personal de GitHub con acceso de lectura al repositorio |

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">Salida</div>

| Parámetro | Tipo | Descripción |
| --------- | ---- | ----------- |
| `repositoryId` | string | Identificador único del repositorio indexado \(formato: remoto:rama:propietario/repo\) |
| `statusEndpoint` | string | Endpoint de URL para verificar el estado de la indexación |
