<div className="text-base font-semibold mb-2">Listar Problemas (<code>sentry_issues_list</code>)</div>

Lista los problemas de Sentry para una organización específica y, opcionalmente, un proyecto específico. Devuelve detalles del problema, incluyendo el estado, el recuento de errores y las marcas de tiempo de la última vez que se detectaron.

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">Entrada</div>

| Parámetro | Tipo | Requerido | Descripción |
| --------- | ---- | -------- | ----------- |
| `apiKey` | string | Sí | Token de autenticación de la API de Sentry |
| `organizationSlug` | string | Sí | El slug de la organización |
| `projectSlug` | string | No | Filtrar problemas por slug de proyecto específico \(opcional\) |
| `query` | string | No | Consulta de búsqueda para filtrar problemas. Admite la sintaxis de búsqueda de Sentry \(p. ej., "is:unresolved", "level:error"\) |
| `statsPeriod` | string | No | Período de tiempo para estadísticas \(p. ej., "24h", "7d", "30d"\). Por defecto, 24h si no se especifica. |
| `cursor` | string | No | Cursor de paginación para recuperar la siguiente página de resultados |
| `limit` | number | No | Número de problemas a devolver por página \(predeterminado: 25, máximo: 100\) |
| `status` | string | No | Filtrar por estado del problema: unresolved, resolved, ignored o muted |
| `sort` | string | No | Orden de clasificación: date, new, freq, priority o user \(predeterminado: date\) |

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">Salida</div>

| Parámetro | Tipo | Descripción |
| --------- | ---- | ----------- |
| `id` | string | ID del proyecto |
| `shortId` | string | Identificador corto del problema |
| `title` | string | Título del problema |
| `culprit` | string | Función o ubicación que causó el problema |
| `permalink` | string | Enlace directo al problema en Sentry |
| `logger` | string | Nombre del logger que reportó el problema |
| `level` | string | Nivel de severidad \(error, warning, info, etc.\) |
| `status` | string | Estado actual del problema |
| `platform` | string | Plataforma donde ocurrió el problema |
| `name` | string | Nombre del proyecto |
| `slug` | string | Slug del proyecto |
