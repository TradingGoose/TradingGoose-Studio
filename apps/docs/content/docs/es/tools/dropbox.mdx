| `mode` | string | No | Modo de escritura: add \(predeterminado\) u overwrite |
| `autorename` | boolean | No | Si es true, renombra el archivo en caso de conflicto |
| `mute` | boolean | No | Si es true, no notifica al usuario sobre esta subida |

| `path_display` | string | Ruta de visualización del archivo |
| `path_lower` | string | Ruta en minúsculas del archivo |
| `size` | number | Tamaño del archivo en bytes |
| `client_modified` | string | Hora de modificación del cliente |
| `server_modified` | string | Hora de modificación del servidor |
| `rev` | string | Identificador de revisión |
