## Herramientas

### `search_tool`

Busca en la web. Devuelve los resultados más relevantes, incluyendo título, enlace, fragmento y fecha para cada resultado.

#### Entrada

| Parámetro | Tipo | Requerido | Descripción |
| --------- | ---- | -------- | ----------- |
| `query` | string | Sí | La consulta de búsqueda |

#### Salida

| Parámetro | Tipo | Descripción |
| --------- | ---- | ----------- |
| `results` | json | Resultados de búsqueda |
| `query` | string | La consulta de búsqueda |
| `totalResults` | number | Número total de resultados |
| `source` | string | Fuente de búsqueda \(exa\) |
| `cost` | json | Información de costo \($0.01\) |

## Instrucciones de uso

Busca en la web utilizando la herramienta de búsqueda. Cada búsqueda tiene un costo de $0.01 por consulta.
