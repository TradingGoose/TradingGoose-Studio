## Herramientas

### `file_parser`

Analiza uno o más archivos cargados o archivos desde URLs (texto, PDF, CSV, imágenes, etc.)

#### Entrada

| Parámetro | Tipo | Requerido | Descripción |
| --------- | ---- | -------- | ----------- |
| `filePath` | string | Sí | Ruta al archivo o archivos. Puede ser una ruta única, URL o un array de rutas. |
| `fileType` | string | No | Tipo de archivo a analizar \(se detecta automáticamente si no se especifica\) |

#### Salida

| Parámetro | Tipo | Descripción |
| --------- | ---- | ----------- |
| `files` | array | Array de archivos analizados con contenido y metadatos |
| `combinedContent` | string | Contenido combinado de todos los archivos analizados |

## Instrucciones de uso

Integra File en el flujo de trabajo. Puede cargar un archivo manualmente o insertar una URL de archivo.
