## Herramientas

### `thinking_tool`

Procesa un pensamiento/instrucción proporcionado, dejándolo disponible para los pasos posteriores.

#### Entrada

| Parámetro | Tipo | Requerido | Descripción |
| --------- | ---- | -------- | ----------- |
| `thought` | string | Sí | Tu razonamiento interno, análisis o proceso de pensamiento. Utilízalo para analizar el problema paso a paso antes de responder. |

#### Salida

| Parámetro | Tipo | Descripción |
| --------- | ---- | ----------- |
| `acknowledgedThought` | string | Proceso de pensamiento reconocido |

## Instrucciones de uso

Añade un paso en el que el modelo describe explícitamente su proceso de razonamiento antes de continuar. Esto puede mejorar la calidad del razonamiento al fomentar un análisis paso a paso.
