<div className="mt-6 border-t border-fd-border pt-4">
<div className="text-base font-semibold mb-2">Realizar Llamada (<code>twilio_voice_make_call</code>)</div>

Realiza una llamada telefónica saliente mediante la API de Twilio Voice.

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">Entrada</div>

| Parámetro | Tipo | Requerido | Descripción |
| --------- | ---- | --------- | ----------- |
| `to` | string | Sí | Número de teléfono a llamar \(formato E.164, p. ej., +14155551234\) |
| `from` | string | Sí | Tu número de teléfono de Twilio desde el que llamar \(formato E.164\) |
| `url` | string | No | URL que devuelve instrucciones TwiML para la llamada |
| `twiml` | string | No | Instrucciones TwiML a ejecutar \(alternativa a la URL\). Usa corchetes en lugar de paréntesis angulares, p. ej., \[Response\]\[Say\]Hello\[/Say\]\[/Response\] |
| `statusCallback` | string | No | URL de webhook para actualizaciones del estado de la llamada |
| `statusCallbackMethod` | string | No | Método HTTP para el callback de estado \(GET o POST\) |
| `accountSid` | string | Sí | SID de cuenta de Twilio |
| `authToken` | string | Sí | Token de autenticación de Twilio |
| `record` | boolean | No | Si se debe grabar la llamada |
| `recordingStatusCallback` | string | No | URL de webhook para actualizaciones del estado de la grabación |
| `timeout` | number | No | Tiempo de espera para respuesta antes de abandonar \(segundos, predeterminado: 60\) |
| `machineDetection` | string | No | Detección de contestador automático: Enable o DetectMessageEnd |

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">Salida</div>

| Parámetro | Tipo | Descripción |
| --------- | ---- | ----------- |
| `success` | boolean | Si la llamada fue iniciada correctamente |
| `callSid` | string | Identificador único de la llamada |
| `status` | string | Estado de la llamada \(queued, ringing, in-progress, completed, etc.\) |
| `direction` | string | Dirección de la llamada \(outbound-api\) |
| `from` | string | Número de teléfono de origen de la llamada |
| `to` | string | Número de teléfono de destino de la llamada |
| `duration` | number | Duración de la llamada en segundos |
| `price` | string | Costo de la llamada |
| `priceUnit` | string | Moneda del precio |

</div>

<div className="text-base font-semibold mb-2">Listar Llamadas (<code>twilio_voice_list_calls</code>)</div>

Recuperar una lista de llamadas realizadas y recibidas en una cuenta.

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">Entrada</div>

| Parámetro | Tipo | Requerido | Descripción |
| --------- | ---- | -------- | ----------- |
| `accountSid` | string | Sí | SID de cuenta de Twilio |
| `authToken` | string | Sí | Token de autenticación de Twilio |
| `to` | string | No | Filtrar por llamadas a este número de teléfono |
| `from` | string | No | Filtrar por llamadas desde este número de teléfono |
| `status` | string | No | Filtrar por estado de llamada \(queued, ringing, in-progress, completed, etc.\) |
| `startTimeAfter` | string | No | Filtrar llamadas iniciadas en o después de esta fecha \(YYYY-MM-DD\) |
| `startTimeBefore` | string | No | Filtrar llamadas iniciadas en o antes de esta fecha \(YYYY-MM-DD\) |
| `pageSize` | number | No | Número de registros a devolver \(máx. 1000, predeterminado 50\) |

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">Salida</div>

| Parámetro | Tipo | Descripción |
| --------- | ---- | ----------- |
| `success` | boolean | Indica si las llamadas fueron recuperadas exitosamente |
| `calls` | array | Arreglo de objetos de llamada |
| `total` | number | Número total de llamadas devueltas |
| `page` | number | Número de página actual |
| `pageSize` | number | Número de llamadas por página |
