## Acceso Directo a la API

Mientras los SDKs están en desarrollo, puede integrarse con TradingGoose utilizando la API REST directamente:

```bash
curl -X POST \
  https://tradinggoose.ai/api/workflows/WORKFLOW_ID/execute \
  -H 'Content-Type: application/json' \
  -H 'X-API-Key: YOUR_KEY' \
  -d '{"input": "your data"}'
```

Consulte la documentación de la [API de Ejecución](/execution/api) para obtener documentación completa de endpoints, streaming y callbacks de webhook.
