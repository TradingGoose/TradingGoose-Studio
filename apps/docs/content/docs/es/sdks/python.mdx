## Estado

El SDK de Python está siendo desarrollado para proporcionar una interfaz Python nativa para ejecutar flujos de trabajo de TradingGoose, gestionar recursos del espacio de trabajo y manejar respuestas en streaming.

## Mientras Tanto

Utiliza la [API de Ejecución](/execution/api) directamente con cualquier cliente HTTP:

```python
import requests

response = requests.post(
    "https://tradinggoose.ai/api/workflows/WORKFLOW_ID/execute",
    headers={
        "Content-Type": "application/json",
        "X-API-Key": "YOUR_KEY",
    },
    json={"input": "your data"},
)

result = response.json()
```

Para respuestas en streaming, consulta la [sección de Streaming](/triggers/api#streaming-responses) de la documentación del API Trigger.
