## Estado

El SDK de TypeScript está siendo desarrollado para proporcionar total seguridad de tipos al ejecutar flujos de trabajo de TradingGoose desde entornos Node.js y de navegador.

## Mientras Tanto

Utiliza la [API de Ejecución](/execution/api) directamente con fetch:

```typescript
const response = await fetch(
  "https://tradinggoose.ai/api/workflows/WORKFLOW_ID/execute",
  {
    method: "POST",
    headers: {
      "Content-Type": "application/json",
      "X-API-Key": "YOUR_KEY",
    },
    body: JSON.stringify({ input: "your data" }),
  }
);

const result = await response.json();
```

Para respuestas en streaming con Server-Sent Events, consulta la [sección de Streaming](/triggers/api#streaming-responses) de la documentación del API Trigger.
