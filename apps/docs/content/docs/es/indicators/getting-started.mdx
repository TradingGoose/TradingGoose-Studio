## Creación de Tu Primer Indicador

<Steps>
  <Step>
    **Abre el Editor de Indicadores** — Añade un widget de Editor de Indicadores a tu espacio de trabajo, o navega a la lista de Indicadores y haz clic en "Nuevo Indicador"
  </Step>
  <Step>
    **Escribe tu código** — Comienza con la declaración `indicator()`, define las entradas, calcula los valores y grafica los resultados
  </Step>
  <Step>
    **Vista previa en el gráfico** — El widget Data Chart renderiza la salida de tu indicador en tiempo real mientras editas
  </Step>
  <Step>
    **Guarda y despliega** — Asigna un nombre a tu indicador y guárdalo en tu espacio de trabajo para usarlo en gráficos y flujos de trabajo
  </Step>
</Steps>

## Ejemplo Mínimo

Todo indicador comienza con una llamada a `indicator()` y al menos un `plot()`:

```javascript
indicator('Simple Moving Average', { overlay: true });

const length = input.int(20, 'Length');
const sma = ta.sma(close, length);

plot(sma, 'SMA', { color: '#2962FF' });
```

Esto crea un indicador con:
- Un parámetro `Length` configurable (valor predeterminado 20) que aparece en la interfaz
- Una media móvil simple calculada sobre los precios de cierre
- Una línea azul graficada como superposición en el gráfico de precios

## Opciones del Indicador

La función `indicator()` acepta un título y un objeto de opciones:

```javascript
indicator('My Indicator', {
  overlay: true,        // Plot on price chart (false = separate pane)
  precision: 2,         // Decimal places for values
  format: format.price, // Display format
});
```

| Opción | Tipo | Valor Predeterminado | Descripción |
|--------|------|---------|-------------|
| `overlay` | boolean | `false` | Graficar en el gráfico de precios en lugar de un panel separado |
| `precision` | number | auto | Número de decimales |
| `format` | string | `format.inherit` | Formato de visualización (`format.price`, `format.volume`, `format.percent`) |
| `scale` | string | auto | Tipo de escala |
| `max_bars_back` | number | auto | Máximo de barras históricas a referenciar |
| `max_lines_count` | number | auto | Máximo de objetos de dibujo de líneas |
| `max_labels_count` | number | auto | Máximo de objetos de dibujo de etiquetas |
| `max_boxes_count` | number | auto | Máximo de objetos de dibujo de cajas |

## Un Ejemplo Más Completo

Aquí hay un indicador de Bandas de Bollinger que demuestra entradas, desestructuración de tuplas, múltiples gráficas y relleno:

```javascript
indicator('Bollinger Bands', { overlay: true });

const length = input.int(20, 'Length');
const mult = input.float(2, 'StdDev');

// ta.bb returns [middle, upper, lower] as a tuple
const [middle, upper, lower] = ta.bb(close, length, mult);

const upperPlot = plot(upper, 'Upper', { color: '#F23645' });
plot(middle, 'Middle', { color: '#2962FF' });
const lowerPlot = plot(lower, 'Lower', { color: '#089981' });

// Fill the area between upper and lower bands
fill(upperPlot, lowerPlot);
```

## Agregar un Disparador de Flujo de Trabajo

La función `trigger()` es una extensión de TradingGoose que conecta las señales de los indicadores con la automatización de flujos de trabajo:

```javascript
indicator('MACD Crossover Signal');

const [macdLine, signalLine, hist] = ta.macd(close, 12, 26, 9);

plot(hist, 'Histogram', { style: plot.style_histogram });
plot(macdLine, 'MACD');
plot(signalLine, 'Signal');

// Fire a workflow when MACD crosses above signal line
if (ta.crossover(macdLine, signalLine)) {
  trigger('macd_bullish_cross', {
    condition: ta.crossover(macdLine, signalLine),
    input: 'MACD bullish crossover detected',
    signal: 'long',
    position: 'belowBar',
    color: '#00FF00'
  });
}
```

<Callout type="info">
  La API `trigger()` es exclusiva de TradingGoose y no existe en Pine Script estándar. Consulta la [guía de Disparadores](/indicators/triggers) para la documentación completa.
</Callout>
