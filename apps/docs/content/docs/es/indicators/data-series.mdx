PineTS proporciona series de datos integradas disponibles en cada indicador sin necesidad de importaciones. Estas representan los datos OHLCV del símbolo actual, información temporal y estado de velas.

## Datos de Precio (OHLCV)

| Serie | Tipo | Descripción |
|--------|------|-------------|
| `open` | `series<number>` | Precio de apertura de cada vela |
| `high` | `series<number>` | Precio máximo de cada vela |
| `low` | `series<number>` | Precio mínimo de cada vela |
| `close` | `series<number>` | Precio de cierre de cada vela |
| `volume` | `series<number>` | Volumen de negociación de cada vela |

### Series de Precio Calculadas

| Serie | Fórmula | Descripción |
|--------|---------|-------------|
| `hl2` | `(high + low) / 2` | Precio punto medio |
| `hlc3` | `(high + low + close) / 3` | Precio típico |
| `ohlc4` | `(open + high + low + close) / 4` | Precio promedio |

```javascript
// Use any price series in calculations
const typicalPrice = hlc3;
const sma = ta.sma(typicalPrice, 20);
plot(sma, 'SMA of Typical Price');
```

## Series Temporales

| Serie | Tipo | Descripción |
|--------|------|-------------|
| `openTime` | `series<number>` | Tiempo de apertura de vela en milisegundos |
| `closeTime` | `series<number>` | Tiempo de cierre de vela en milisegundos |

## Información de Vela

| Variable | Tipo | Descripción |
|----------|------|-------------|
| `bar_index` | `series<number>` | Índice de la vela actual (base 0) |
| `last_bar_index` | `series<number>` | Índice de la última vela en el conjunto de datos |
| `last_bar_time` | `series<number>` | Marca de tiempo de la última vela |

```javascript
// Only plot on the last 100 bars
if (bar_index > last_bar_index - 100) {
  plot(close, 'Recent Close');
}
```

## Estado de Vela

El objeto `barstate` proporciona indicadores sobre la posición de la vela actual en el conjunto de datos:

| Propiedad | Tipo | Descripción |
|----------|------|-------------|
| `barstate.isfirst` | `boolean` | Verdadero en la primera vela |
| `barstate.islast` | `boolean` | Verdadero en la última vela |
| `barstate.ishistory` | `boolean` | Verdadero para velas históricas |
| `barstate.isrealtime` | `boolean` | Verdadero para velas en vivo/streaming |
| `barstate.isnew` | `boolean` | Verdadero en el primer tick de una nueva vela |
| `barstate.isconfirmed` | `boolean` | Verdadero cuando la vela está confirmada (cerrada) |
| `barstate.islastconfirmedhistory` | `boolean` | Verdadero en la última vela histórica confirmada |

## Información del Símbolo

El objeto `syminfo` proporciona detalles sobre el símbolo actual:

| Propiedad | Descripción |
|----------|-------------|
| `syminfo.ticker` | Nombre del símbolo (p. ej., `"BTCUSDT"`) |
| `syminfo.tickerid` | ID completo del ticker |
| `syminfo.description` | Descripción del símbolo |
| `syminfo.type` | Tipo de activo (p. ej., `"crypto"`, `"stock"`) |
| `syminfo.currency` | Moneda de cotización |
| `syminfo.basecurrency` | Moneda base |
| `syminfo.mintick` | Incremento mínimo de precio |
| `syminfo.timezone` | Zona horaria del exchange |

## Valores Especiales

| Valor | Descripción |
|-------|-------------|
| `na` | Representa "no disponible" — similar a `null`. Usar para verificar si un valor existe |
| `nz(value, replacement?)` | Devuelve `replacement` (`0` por defecto) si `value` es `na` |

```javascript
const prev = close[1];
// na check
if (!na(prev)) {
  const change = close - prev;
  plot(change, 'Change');
}

// nz shorthand — default to 0 if na
const safeChange = nz(close - close[1], 0);
```

## Constantes Globales

| Espacio de nombres | Descripción |
|-----------|-------------|
| `timeframe` | Información del timeframe actual (`timeframe.period`, `timeframe.multiplier`) |
| `order` | Constantes de orden de clasificación (`order.ascending`, `order.descending`) |
| `currency` | Constantes de códigos de divisa (59 divisas incluyendo `currency.BTC`, `currency.USD`) |
| `display` | Constantes de modo de visualización |
| `shape` | Constantes de tipo de figura para `plotshape()` |
| `location` | Constantes de ubicación (`location.abovebar`, `location.belowbar`, etc.) |
| `size` | Constantes de tamaño (`size.tiny`, `size.small`, `size.normal`, `size.large`, `size.huge`) |
| `format` | Constantes de formato (`format.price`, `format.volume`, `format.percent`) |
| `dayofweek` | Constantes de día de la semana |

<Callout type="info">
  Todas las series de datos admiten acceso al historial con notación de corchetes: `close[1]` devuelve el cierre de la vela anterior, `close[10]` devuelve 10 velas atrás.
</Callout>
