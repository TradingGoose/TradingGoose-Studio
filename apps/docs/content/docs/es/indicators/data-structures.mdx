## Array (52 métodos)

Los arrays son colecciones ordenadas de elementos. Cree, manipule y consulte arrays para implementar lógica avanzada en indicadores.

### Creación

```javascript
const a = array.from(1, 2, 3, 4, 5);
const b = array.new_float(10, 0.0);    // 10 elements, all 0.0
const c = array.new_int(5, 0);
const d = array.new_bool(3, false);
const e = array.new_string(2, '');
```

### Acceso y Modificación

| Función | Descripción |
|----------|-------------|
| `array.get(arr, index)` | Obtener elemento en el índice |
| `array.set(arr, index, value)` | Establecer elemento en el índice |
| `array.first(arr)` | Obtener el primer elemento |
| `array.last(arr)` | Obtener el último elemento |
| `array.push(arr, value)` | Agregar al final |
| `array.pop(arr)` | Eliminar y retornar el último |
| `array.unshift(arr, value)` | Insertar al inicio |
| `array.shift(arr)` | Eliminar y retornar el primero |
| `array.insert(arr, index, value)` | Insertar en el índice |
| `array.remove(arr, index)` | Eliminar en el índice |
| `array.clear(arr)` | Eliminar todos los elementos |

### Tamaño y Transformación

| Función | Descripción |
|----------|-------------|
| `array.size(arr)` | Número de elementos |

### Búsqueda

|----------|-------------|

## Mapa (11 métodos)

Diccionarios clave-valor para datos asociativos:

```javascript
const m = map.new();
map.put(m, 'ema', ta.ema(close, 20));
map.put(m, 'sma', ta.sma(close, 20));

const emaValue = map.get(m, 'ema');
const hasKey = map.contains(m, 'ema');     // true
const allKeys = map.keys(m);               // ['ema', 'sma']
```

| Función | Descripción |
|----------|-------------|
| `map.new()` | Crear mapa vacío |
| `map.get(m, key)` | Obtener valor |
| `map.put(m, key, value)` | Establecer valor |
| `map.put_all(m, other)` | Copiar todo desde otro mapa |
| `map.contains(m, key)` | Verificar si existe la clave |
| `map.remove(m, key)` | Eliminar clave |
| `map.keys(m)` | Todas las claves |
| `map.values(m)` | Todos los valores |
| `map.size(m)` | Número de elementos |
| `map.clear(m)` | Eliminar todo |
| `map.copy(m)` | Duplicar |

## Matriz (50 métodos)

Estructuras de datos 2D para operaciones matemáticas avanzadas:

```javascript
const m = matrix.new(3, 3, 0);  // 3x3 matrix of zeros
matrix.set(m, 0, 0, 1);
matrix.set(m, 1, 1, 1);
matrix.set(m, 2, 2, 1);         // Identity matrix
```

### Operaciones clave

| Categoría | Funciones |
|----------|-----------|
| **Acceso** | `get`, `set`, `row`, `col` |
| **Tamaño** | `rows`, `columns`, `elements_count` |
| **Modificar** | `add_row`, `add_col`, `remove_row`, `remove_col`, `swap_rows`, `swap_columns`, `fill`, `reverse` |
| **Transformar** | `transpose`, `reshape`, `slice`, `submatrix`, `concat`, `copy` |
| **Aritmética** | `mult`, `pow`, `diff` |
| **Álgebra lineal** | `det`, `inv`, `pinv`, `eigenvalues`, `eigenvectors`, `rank`, `trace` |
| **Estadísticas** | `sum`, `avg`, `min`, `max`, `median`, `mode`, `stdev`, `variance` |
| **Pruebas booleanas** | `is_square`, `is_identity`, `is_diagonal`, `is_symmetric`, `is_triangular`, `is_zero`, `is_binary`, `is_stochastic` |
