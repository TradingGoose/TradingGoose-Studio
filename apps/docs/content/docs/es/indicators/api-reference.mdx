<Callout type="info">
  Esta referencia cubre las funciones disponibles en el entorno de ejecución PineTS de TradingGoose. PineTS está basado en el proyecto de código abierto [PineTS project](https://github.com/nicholasgasior/PineTS) y admite un subconjunto de las APIs de Pine Script v5/v6. Funcionalidades como `strategy.*`, `label.*`, `line.*`, `box.*` y `table.*` no están soportadas actualmente.
</Callout>
