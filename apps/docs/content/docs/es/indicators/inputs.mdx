El espacio de nombres `input` le permite definir parámetros que los usuarios pueden configurar en la interfaz sin necesidad de editar el código. TradingGoose genera automáticamente un panel de configuración a partir de sus llamadas `input.*()`.

## Tipos de entrada

| Función | Retorna | Descripción |
|----------|---------|-------------|
| `input.int(defval, title, minval?, maxval?, step?)` | `number` | Entrada entera con rango y paso opcionales |
| `input.float(defval, title, minval?, maxval?, step?)` | `number` | Entrada de punto flotante |
| `input.bool(defval, title)` | `boolean` | Interruptor/casilla de verificación |
| `input.string(defval, title)` | `string` | Entrada de texto |
| `input.color(defval, title)` | `string` | Selector de color (cadena hexadecimal) |
| `input.source(defval, title)` | `series` | Selector de fuente de datos (close, open, high, low, etc.) |
| `input.timeframe(defval, title)` | `string` | Selector de marco temporal |
| `input.price(defval, title)` | `number` | Entrada de nivel de precio |
| `input.time(defval, title)` | `number` | Entrada de marca de tiempo |
| `input.session(defval, title)` | `string` | Entrada de sesión de trading |
| `input.symbol(defval, title)` | `string` | Entrada de símbolo/ticker |
| `input.text_area(defval, title)` | `string` | Entrada de texto multilínea |
| `input.enum(defval, title, options)` | `T` | Selector desplegable de una lista de opciones |

## Ejemplos

### Entradas numéricas

```javascript
// Integer with range constraints
const length = input.int(14, 'RSI Length', 1, 200, 1);

// Float with step
const multiplier = input.float(2.0, 'StdDev Multiplier', 0.1, 10.0, 0.1);
```

### Booleano y cadena de texto

```javascript
const showSignals = input.bool(true, 'Show Signals');
const customLabel = input.string('Buy', 'Signal Label');
```

### Selección de fuente

```javascript
// Let users choose which price data to calculate on
const src = input.source(close, 'Source');
const sma = ta.sma(src, 20);
```

### Enum (desplegable)

```javascript
const maType = input.enum('SMA', 'MA Type', ['SMA', 'EMA', 'WMA', 'HMA']);

let ma;
switch (maType) {
  case 'EMA': ma = ta.ema(close, length); break;
  case 'WMA': ma = ta.wma(close, length); break;
  case 'HMA': ma = ta.hma(close, length); break;
  default:    ma = ta.sma(close, length);
}
```

### Entrada de color

```javascript
const bullColor = input.color('#00FF00', 'Bull Color');
const bearColor = input.color('#FF0000', 'Bear Color');

barcolor(close > open ? bullColor : bearColor);
```

## Cómo funcionan las entradas

Cuando TradingGoose carga su indicador, analiza todas las llamadas `input.*()` para:

1. **Extraer metadatos** — título, tipo, valor por defecto, restricciones (min/max/paso/opciones)
2. **Generar controles de interfaz** — renderiza los controles de formulario apropiados en el panel de configuración del indicador
3. **Pasar valores en tiempo de ejecución** — cuando un usuario modifica un parámetro, el indicador se vuelve a ejecutar con los valores actualizados

<Callout type="info">
  Los parámetros de entrada se extraen de forma estática desde tu código. Mantén las llamadas a `input.*()` en el nivel superior de tu script para obtener mejores resultados — evita colocarlas dentro de condicionales o bucles.
</Callout>
