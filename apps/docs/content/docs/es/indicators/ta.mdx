El espacio de nombres `ta` proporciona más de 57 funciones de análisis técnico que cubren medias móviles, osciladores, volatilidad, volumen, detección de tendencias y estadísticas.

## Medias Móviles

| Función | Descripción |
|----------|-------------|
| `ta.sma(source, length)` | Media Móvil Simple |
| `ta.ema(source, length)` | Media Móvil Exponencial |
| `ta.wma(source, length)` | Media Móvil Ponderada |
| `ta.vwma(source, length)` | Media Móvil Ponderada por Volumen |
| `ta.rma(source, length)` | Media Móvil Continua/Acumulada (suavizado de Wilder) |
| `ta.hma(source, length)` | Media Móvil Hull |
| `ta.alma(source, length, offset?, sigma?)` | Media Móvil Arnaud Legoux |
| `ta.swma(source)` | Media Móvil Simétricamente Ponderada |
| `ta.linreg(source, length, offset?)` | Regresión Lineal |
| `ta.vwap` | Precio Promedio Ponderado por Volumen (variable) |
| `ta.vwap(source)` | Precio Promedio Ponderado por Volumen (función) |

```javascript
const fast = ta.ema(close, 9);
const slow = ta.ema(close, 21);
plot(fast, 'Fast EMA', { color: '#2962FF' });
plot(slow, 'Slow EMA', { color: '#FF6D00' });
```

## Osciladores y Momento

| Función | Retorna | Descripción |
|----------|---------|-------------|
| `ta.rsi(source, length)` | `number` | Índice de Fuerza Relativa |
| `ta.macd(source, fast, slow, signal)` | `[macd, signal, hist]` | MACD (retorna tupla) |
| `ta.stoch(source, high, low, length)` | `number` | Oscilador Estocástico |
| `ta.cci(source, length)` | `number` | Índice de Canal de Materias Primas |
| `ta.mfi(series, length)` | `number` | Índice de Flujo de Dinero |
| `ta.mom(source, length)` | `number` | Momento |
| `ta.roc(source, length)` | `number` | Tasa de Cambio |
| `ta.tsi(source, short, long)` | `number` | Índice de Fuerza Verdadera |
| `ta.cmo(source, length)` | `number` | Oscilador de Momento Chande |
| `ta.cog(source, length)` | `number` | Centro de Gravedad |
| `ta.wpr(length)` | `number` | Williams %R |
| `ta.change(source, length?)` | `number` | Variación de precio en N barras |

```javascript
const rsi = ta.rsi(close, 14);
const [macdLine, signalLine, hist] = ta.macd(close, 12, 26, 9);
```

## Volatilidad y Rango

| Función | Retorna | Descripción |
|----------|---------|-------------|
| `ta.atr(length)` | `number` | Rango Verdadero Promedio (ATR) |
| `ta.bb(source, length, mult)` | `[middle, upper, lower]` | Bandas de Bollinger (tupla) |
| `ta.bbw(source, length, mult)` | `number` | Ancho de Bandas de Bollinger |
| `ta.kc(source, length, mult)` | `[middle, upper, lower]` | Canales de Keltner (tupla) |
| `ta.kcw(source, length, mult)` | `number` | Ancho de Canales de Keltner |
| `ta.stdev(source, length)` | `number` | Desviación Estándar |
| `ta.dev(source, length)` | `number` | Desviación Absoluta Media |
| `ta.variance(source, length)` | `number` | Varianza |
| `ta.range(source, length)` | `number` | Rango (máximo - mínimo) |
| `ta.tr` | `number` | Rango Verdadero (variable) |
| `ta.tr(handleNa?)` | `number` | Rango Verdadero (función) |

```javascript
const [middle, upper, lower] = ta.bb(close, 20, 2);
const atr = ta.atr(14);
```

## Indicadores de Volumen

Se accede a estos como variables (no como llamadas a función):

| Variable | Descripción |
|----------|-------------|
| `ta.obv` | Volumen en Balance (OBV) |
| `ta.accdist` | Acumulación/Distribución |
| `ta.pvt` | Tendencia Precio-Volumen |
| `ta.wad` | Acumulación/Distribución de Williams |
| `ta.wvad` | Acumulación/Distribución Variable de Williams |
| `ta.nvi` | Índice de Volumen Negativo |
| `ta.pvi` | Índice de Volumen Positivo |
| `ta.iii` | Índice de Intensidad Intradía |

```javascript
plot(ta.obv, 'OBV');
plot(ta.accdist, 'A/D');
```

## Análisis de Tendencia

| Función | Retorna | Descripción |
|----------|---------|-------------|
| `ta.crossover(a, b)` | `boolean` | Verdadero cuando `a` cruza por encima de `b` |
| `ta.crossunder(a, b)` | `boolean` | Verdadero cuando `a` cruza por debajo de `b` |
| `ta.cross(a, b)` | `boolean` | Verdadero cuando `a` cruza `b` en cualquier dirección |
| `ta.rising(source, length)` | `boolean` | Verdadero cuando la fuente ha estado subiendo durante `length` barras |
| `ta.falling(source, length)` | `boolean` | Verdadero cuando la fuente ha estado bajando durante `length` barras |
| `ta.supertrend(factor, atrLength)` | `[value, direction]` | SuperTrend (tupla) |
| `ta.sar(start, inc, max)` | `number` | SAR Parabólico |
| `ta.dmi(diLength, adxSmoothing)` | `[plus, minus, adx]` | Índice de Movimiento Direccional (tupla) |

```javascript
if (ta.crossover(fast, slow)) {
  plotshape(close, 'Buy', { style: shape.triangleup, location: location.belowbar });
}
```
