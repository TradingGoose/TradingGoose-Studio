| `limit` | number | No | Number of issues to return per page \(default: 25, max: 100\) |
| `status` | string | No | Filter by issue status: unresolved, resolved, ignored, or muted |
| `sort` | string | No | Sort order: date, new, freq, priority, or user \(default: date\) |

<div className="text-sm font-medium text-fd-muted-foreground mt-4 mb-2">Output</div>

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `id` | string | Project ID |
| `shortId` | string | Short issue identifier |
| `title` | string | Issue title |
| `culprit` | string | Function or location that caused the issue |
| `permalink` | string | Direct link to the issue in Sentry |
| `logger` | string | Logger name that reported the issue |
