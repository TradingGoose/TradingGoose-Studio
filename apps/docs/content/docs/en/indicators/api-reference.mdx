<Callout type="info">
  This reference covers functions available in TradingGoose's PineTS runtime. PineTS is based on the open-source [PineTS project](https://github.com/nicholasgasior/PineTS) and supports a subset of Pine Script v5/v6 APIs. Features like `strategy.*`, `label.*`, `line.*`, `box.*`, and `table.*` are not currently supported.
</Callout>
