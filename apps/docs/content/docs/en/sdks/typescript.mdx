## Status

The TypeScript SDK is being built to provide full type safety for executing TradingGoose workflows from Node.js and browser environments.

## In the Meantime

Use the [Execution API](/execution/api) directly with fetch:

```typescript
const response = await fetch(
  "https://tradinggoose.ai/api/workflows/WORKFLOW_ID/execute",
  {
    method: "POST",
    headers: {
      "Content-Type": "application/json",
      "X-API-Key": "YOUR_KEY",
    },
    body: JSON.stringify({ input: "your data" }),
  }
);

const result = await response.json();
```

For streaming responses with Server-Sent Events, see the [Streaming section](/triggers/api#streaming-responses) of the API Trigger docs.
