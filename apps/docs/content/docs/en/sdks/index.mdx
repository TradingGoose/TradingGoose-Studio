## Direct API Access

While SDKs are in development, you can integrate with TradingGoose using the REST API directly:

```bash
curl -X POST \
  https://tradinggoose.ai/api/workflows/WORKFLOW_ID/execute \
  -H 'Content-Type: application/json' \
  -H 'X-API-Key: YOUR_KEY' \
  -d '{"input": "your data"}'
```

See the [Execution API](/execution/api) docs for complete endpoint documentation, streaming, and webhook callbacks.
