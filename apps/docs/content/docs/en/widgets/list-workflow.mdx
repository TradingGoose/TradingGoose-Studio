## Overview

| Property | Value |
|----------|-------|
| **Key** | `list_workflow` |
| **Category** | List |

The Workflow List widget displays a browseable list of all workflows in your workspace.

## Features

- **Browse**: View all workflows in your workspace
- **Search & Filter**: Find workflows quickly
- **Create New**: Start a new workflow directly from the list
- **Open in Editor**: Click to open any workflow in the editor widget

## Usage

Add this widget alongside the Workflow Editor to manage your workflows.
