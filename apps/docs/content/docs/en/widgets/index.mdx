Widgets are the dashboard UI components that make up your workspace. Each widget occupies a panel in the workspace layout and provides a focused interface for a specific task — editing a workflow, monitoring execution, charting market data, or managing workspace resources. You can rearrange, resize, and swap widgets freely to build a layout that fits your needs.
