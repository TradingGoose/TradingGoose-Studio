The Workflow Copilot widget provides an AI-powered assistant experience specifically designed to help you build and optimize your workflows.

## Overview

| Property | Value |
|----------|-------|
| **Key** | `workflow_copilot` |
| **Category** | Utility |
| **Icon** | Sparkles |

## Features

- **AI-Powered Assistance**: Get intelligent help for workflow design
- **Context-Aware**: Understands your current workflow structure
- **Block Suggestions**: Recommends blocks based on your goals
- **Troubleshooting**: Help diagnose and fix workflow issues
- **Best Practices**: Suggestions for optimizing your workflows

## Usage

The Workflow Copilot can help you with:

- Designing new workflows from scratch
- Optimizing existing workflows
- Understanding block configurations
- Debugging execution issues
- Learning TradingGoose features

### Example Interactions

**Creating a new workflow:**
> "Create a workflow that monitors a GitHub repository and sends Slack notifications for new issues"

**Understanding blocks:**
> "What does the Router block do and when should I use it?"

**Debugging:**
> "Why is my workflow failing at the API block?"

<Callout>
This widget supports the **pairColor** mechanism. Pair it with a Workflow Editor to get assistance for the specific workflow you're working on.
</Callout>
