The Workflow List widget displays all your workflows in a hierarchical folder tree, allowing you to organize, navigate, and manage your workflow collection.

## Overview

| Property | Value |
|----------|-------|
| **Key** | `workflow_list` |
| **Category** | List |
| **Icon** | ListTree |

## Features

- **Folder Tree**: Hierarchical organization of workflows
- **Drag-and-Drop**: Move workflows between folders
- **Quick Navigation**: Click to open workflows
- **Context Menu**: Right-click for additional actions
- **Search**: Filter workflows by name
- **Create New**: Add new workflows and folders

## Usage

The Workflow List widget mirrors the sidebar navigation and provides a dedicated view for managing your workflows.

### Organization

- **Folders**: Create folders to group related workflows
- **Drag-and-Drop**: Move items by dragging them to new locations
- **Rename**: Double-click or use context menu to rename
- **Delete**: Remove workflows or empty folders

### Context Menu Actions

Right-click on a workflow or folder to access:

| Action | Description |
|--------|-------------|
| **Open** | Open the workflow in the editor |
| **Rename** | Change the name |
| **Duplicate** | Create a copy |
| **Move to** | Move to another folder |
| **Delete** | Remove the item |

### Selecting Workflows

Click on a workflow to select it. When paired with other widgets using the same color, the selected workflow will be displayed in those widgets.

<Callout>
This widget supports the **pairColor** mechanism. When you select a workflow in this list, other paired widgets will automatically switch to show that workflow.
</Callout>
