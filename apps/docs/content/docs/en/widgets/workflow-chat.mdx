The Workflow Chat widget provides an interactive chat interface for communicating with your workflow blocks. It enables conversational interactions with your AI workflows.

## Overview

| Property | Value |
|----------|-------|
| **Key** | `workflow_chat` |
| **Category** | Utility |
| **Icon** | MessageCircle |

## Features

- **Interactive Chat**: Send messages and receive responses from your workflow
- **Output Selection**: Choose which workflow outputs to display in the chat
- **File Upload**: Attach files to your messages for processing
- **Real-time Streaming**: See responses as they are generated
- **Message History**: View and scroll through conversation history

## Usage

The Workflow Chat widget is useful for:

- Testing workflows that have chat-based triggers
- Interacting with AI agents in your workflow
- Debugging workflow responses
- Demonstrating workflow capabilities

### Sending Messages

1. Type your message in the input field
2. Optionally attach files using the upload button
3. Press Enter or click Send to submit
4. View the streaming response in real-time

### Output Selection

You can select which block outputs to display in the chat response. This is helpful when your workflow has multiple output blocks and you want to focus on specific results.

<Callout>
This widget supports the **pairColor** mechanism. Pair it with a Workflow Editor to chat with the workflow you're currently editing.
</Callout>
