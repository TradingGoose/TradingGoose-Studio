Le widget Workflow Copilot fournit une expérience d'assistant alimentée par l'IA spécialement conçue pour vous aider à construire et optimiser vos workflows.

## Aperçu

| Propriété | Valeur |
|----------|-------|
| **Clé** | `workflow_copilot` |
| **Catégorie** | Utilitaire |
| **Icône** | Étincelles |

## Fonctionnalités

- **Assistance alimentée par l'IA** : obtenez une aide intelligente pour la conception de workflow
- **Sensible au contexte** : comprend la structure actuelle de votre workflow
- **Suggestions de blocs** : recommande des blocs en fonction de vos objectifs
- **Dépannage** : aide à diagnostiquer et résoudre les problèmes de workflow
- **Meilleures pratiques** : suggestions pour optimiser vos workflows

## Utilisation

Le Workflow Copilot peut vous aider avec :

- La conception de nouveaux workflows à partir de zéro
- L'optimisation des workflows existants
- La compréhension des configurations de blocs
- Le débogage des problèmes d'exécution
- L'apprentissage des fonctionnalités de TradingGoose

### Exemples d'interactions

**Création d'un nouveau workflow :**
> "Créer un workflow qui surveille un dépôt GitHub et envoie des notifications Slack pour les nouveaux problèmes"

**Comprendre les blocs :**
> "Que fait le bloc Router et quand devrais-je l'utiliser ?"

**Débogage :**
> "Pourquoi mon workflow échoue-t-il au niveau du bloc API ?"

<Callout>
Ce widget prend en charge le mécanisme **pairColor**. Associez-le à un éditeur de workflow pour obtenir de l'aide sur le workflow spécifique sur lequel vous travaillez.
</Callout>
