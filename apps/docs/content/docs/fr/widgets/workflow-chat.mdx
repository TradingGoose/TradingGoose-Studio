Le widget Chat de workflow offre une interface de chat interactive pour communiquer avec vos blocs de workflow. Il permet des interactions conversationnelles avec vos workflows IA.

## Vue d’ensemble

| Propriété | Valeur |
|-----------|--------|
| **Clé** | `workflow_chat` |
| **Catégorie** | Utilitaire |
| **Icône** | MessageCircle |

## Fonctionnalités

- **Chat interactif** : Envoyez des messages et recevez des réponses de votre workflow
- **Sélection de sortie** : Choisissez quelles sorties de workflow afficher dans le chat
- **Envoi de fichiers** : Joignez des fichiers à vos messages pour traitement
- **Diffusion en temps réel** : Voyez les réponses au fur et à mesure de leur génération
- **Historique des messages** : Consultez et faites défiler l’historique de conversation

## Utilisation

Le widget Chat de workflow est utile pour :

- Tester des workflows avec des déclencheurs basés sur le chat
- Interagir avec des agents IA dans votre workflow
- Déboguer les réponses du workflow
- Démontrer les capacités du workflow

### Envoi de messages

1. Saisissez votre message dans le champ de saisie
2. Joignez éventuellement des fichiers via le bouton de téléversement
3. Appuyez sur Entrée ou cliquez sur Envoyer pour soumettre
4. Regardez la réponse diffusée en temps réel

### Sélection de sortie

Vous pouvez choisir quelles sorties de blocs afficher dans la réponse du chat. C’est utile lorsque votre workflow comporte plusieurs blocs de sortie et que vous voulez vous concentrer sur des résultats spécifiques.

<Callout>
Ce widget prend en charge le mécanisme **pairColor**. Associez-le à un éditeur de workflow pour discuter avec le workflow que vous êtes en train de modifier.
</Callout>
