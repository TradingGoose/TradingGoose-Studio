Le widget Variables de flux de travail fournit une interface dédiée pour visualiser et modifier les variables définies dans votre flux de travail.

## Aperçu

| Propriété | Valeur |
|----------|-------|
| **Clé** | `workflow_variables` |
| **Catégorie** | Utilitaire |
| **Icône** | Accolades |

## Fonctionnalités

- **Liste des variables** : visualisez toutes les variables du flux de travail en un coup d'œil
- **Édition en ligne** : modifiez directement les valeurs des variables
- **Affichage du type** : consultez le type de données de chaque variable
- **Ajout/Suppression** : créez de nouvelles variables ou supprimez celles existantes
- **Recherche** : filtrez les variables par nom

## Utilisation

Le widget Variables de flux de travail est utile pour :

- Inspecter l'état actuel des variables du flux de travail
- Modifier les valeurs des variables pour les tests
- Comprendre le flux de données dans votre flux de travail
- Gérer la configuration à portée du flux de travail

### Types de variables

Les variables de flux de travail peuvent contenir différents types de données :

| Type | Description |
|------|-------------|
| **Chaîne** | Valeurs textuelles |
| **Nombre** | Valeurs numériques |
| **Booléen** | Valeurs vrai/faux |
| **Objet** | Objets JSON |
| **Tableau** | Listes de valeurs |

### Modification des variables

1. Cliquez sur une variable pour la sélectionner
2. Modifiez la valeur dans l'éditeur
3. Les modifications sont enregistrées automatiquement
4. Le flux de travail utilise la valeur mise à jour lors de la prochaine exécution

### Création de variables

1. Cliquez sur le bouton "Ajouter une variable"
2. Saisissez un nom pour la variable
3. Sélectionnez le type de variable
4. Définissez une valeur initiale (facultatif)

<Callout>
Ce widget prend en charge le mécanisme **pairColor**. Associez-le à un Éditeur de flux de travail pour visualiser les variables du flux de travail que vous êtes en train de modifier.
</Callout>
